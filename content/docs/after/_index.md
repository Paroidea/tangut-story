# 后记

关于西夏的知识，以往主要依赖汉文资料。
如今，围绕出土的各种文物，并通过解读大量保存下来的西夏语文献，相关研究正在推进。
西夏文字的分析和意义解读，也因此成为这一目标的基本工作。

这项工作不仅限于解释每个单字的意义，作为词语的意义、作为熟语的认定也变得重要起来。
西夏人编纂的《（三才）杂字》这一词汇集的复原，已经在笔者手中完成，它在这项工作中起着一种基础资料的作用。
然而，在多种语境中，各个文字以及词语究竟如何使用、表达什么含义，这类信息越多，对阐明这种语言文字的贡献就越大，这是毫无疑问的。
因此，编纂一部记录这些信息的辞书，已成为必不可少的要求。

第四章所涉及的格言、金言研究，也作为其中一个环节，具有重要意义。

在留存下来的西夏语文献中，有像苏联的库恰诺夫正在研究的《天盛旧改新定律令》那样，传达了西夏全盛时期律令的全貌的文献；
也有《圣立义海》这样值得关注的、关于西夏知识的宝库。
该书的解说与翻译目前正在进行中，笔者认为必须尽早完成。

这里提到的书物，不过是一些具有代表性的例子之一。
西夏研究中仍有许多尚未解读的文书和尚未开拓的领域，而且今后还可能出现怎样的资料也尚未可知，这正是其中巨大的期待所在。
