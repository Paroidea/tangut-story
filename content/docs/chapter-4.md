# 第四章 西夏人的格言

中国的少数民族，大多数都喜欢格言。
西夏人也不例外，历史上留下了许多格言。
所谓格言，也就是先人遗留下来的语句，也反映了该民族的传统思想。
因此，其中也有一些在当代的一般常识下不容易理解的地方。

接下来，我会任意选取一些西夏人的格言，尝试从个人理解和文字分析的角度进行说明。
不过，由于这些格言的语句都比较简洁，如果不补充一些额外的文字的话，是很难把握其准确意义的。
西夏语原文全部选自著名西夏学者梁德育编纂的《新集锦合道理》（公元1188年）。

青海即便不深，姐妹内心要纯洁。”

“黑山”（固有名）是一座高耸的山，即使它不再高了，兄弟的约定也应当是坚固的。
青海是深湖，但哪怕它不再深了，姐妹也应当保持清白。

原文中的“𗄽”（山），与“𘑗”一样，像前面提到的那样是汉语中“山”的借用词，读音与汉语相同，为“山”。
第二个字是“黑”，加上一个冠，就成为了第二句里的“深”，是因为深的地方看上去会是黑色的。
此外，“黑”和“深”的发音非常接近。
西夏文字中的“𗗚”（海），不像汉字那样带水旁，其字形来源不明。
“高”的含义，是通过单体字“上”来表示的。

在西夏的文物中还保留有一些印章。
在印章的把手上会刻有这一文字“𗹭”。
另外，西夏人称自己的国家为“白上大国”，其中“上”的字就是这个字。

“兄弟”两字（𗤾𘈑）都是由“𘈑”（人）和“人”派生得到的。
而“姐妹”两字（𘗨𘚋）则是由“𗕪”（女性）派生的文字，文字的旁（𘩭）与“𗤳”（姓）相同。
“坚”字（𗬢）表示“坚固”（𗬢𗸁）的意思，根据西夏人的分析，是由“𗬝”（乌龟）和“𗲺”（坚固）的旁所构成的。

此外还有两个字“𗔲”、“𗲺”，也都是“坚固”的意思。

“𗑗”字（清净），是由“𗄾”（算）作为音符组成的形声字，发音为“腮”。
其顶上的冠与“𗒘”（真实、纯真）是一样的。

## 例2

“即便绫锦不再美丽，结婚也是美好的；
即便兵器不再锋利，朋友关系也是坚固的”

绫和锦都是美丽的东西，但与之相比婚姻是更加美好的。
即使刀具不锋利，朋友之间的关系也会像铠甲一样牢固，不会破裂。

“绫”和“锦”都是与“布”有关的字形，“绫”（𗭄）中间是“美”（𘆡），而“锦”（𗬻）中间则是“妙”（𗤓）的旁。
两者都是与中间是“虫”偏的“丝绢”（𗬼）存在关联。

“色彩很好”（𗌳）是由“花”（𗤻）和“颜色”（𗩨）构成，
表示像花的颜色一样美丽的色彩。
“𘄋”是由“男子”（𘄄）与“娶”（𗀟）组成的文字，而“𘗧”则是由“女性”（𗕪）和“嫁”所构成的。
“𗡮”（美丽）是与“𗵽”（彩色）类似，只有偏有所不同的文字。

{{< tangut_highlight 
  line1="𗩖𗾱,𗕂𗏕"
  line2="武器,武器"
  title="图4 表示“武器”的两种字形"
>}}

“𗩖𗾱”（道具、武器）中的第一个字是“𗦳”（主）与第二个字的除去偏的部分组成的文字。
“用于防卫的工具”即是“武器”的意思。
此外还有一个非常类似的字形“𗕂𗏕”也表示“武器”（图4）。
其中的第二个字“𗏕”与“𗬝”（龟）读音相同，是在“龟”字的偏的基础上创造出来的。
这两个同义的单词，或许反映了西夏文中词汇层I和词汇层II之间的对立关系。

“𘗁”（锋利、锐利）的字形，是“金”冠和“优越、精于”（𗹠）组合而成的，表示“切起来很顺利”的意思。
它在含义上与“𗅰”（钝）是对立的，在佛典中有“𘗁𗅰”（利钝）的用法。
“钝”的字形，则是由“𘎭”（敏）的否定形式与“𘙊”（怠）的旁所组成的。
而直接对“敏”进行否定，也能得到的文字“𗆒”（懈）。

翻译为“朋友”的两个文字，虽然西夏人对其背后的含义进行了分析，但是理解起来比较困难，因此这里就不具体展开了。

最后的翻译为“坚”的文字“𘙦”，原本是“龟”的含义，在这里应该指的是龟的甲壳。
这个文字的字形，原本带的是“皮”偏，将其替换为“水”偏（𗌝），就表示“鳖”的含义。
这句话可以理解为“真正的朋友之间的关系，应该像龟或是鳖的加壳那样坚硬，即便是锐利的兵器也难以将其刺破”的含义。

如果鹫在很高的地方，要想射中它的话，箭矢上凸起的的羽毛就会成为阻碍；
如果鱼在很深的水中游动，普通的钓线就会显得太短，钓不到鱼（原文见图5）。

处理事情时应当根据对方的情况顺应变化，适当更换工具，否则难以达成目的。

“箭”字的旁是由“手”偏构成的，而加上“金”冠之后，得到的派生字“𘖼”表示“带有箭头的箭”的意思。
表示“羽毛”的字是由“鸟”偏和“鼻”构成的。
而“翼”字的写法是“𘝁”，
前半句最后一个字表示“飞出、超出”，表示超越通常的状态。

西夏语中有很多种存在动词。
这些存在动词各有各的字形，其中很多词的具体含义目前人们还没有完全理解。
其中一个动词是第二句中出现的“𗋐”。
虽然不知道这个字为何具有“水”偏，但可以认为它表示的是“附属于容器中的某种东西的事物”的意思。

“钓”字，原本表示“钩”的意思，是由“铁”（𘟪）与“爪”（𘁯）两个字组合而成的。

## 例4

“即使挤压干枯的沙子，也挤不出水；
即使过滤清水，也不会浮起杂质。”

  line3="即使挤压干枯的沙子，也挤不出水来,即使过滤清水，也不会浮起杂质"
  cspan1="7,7"

  cspan3="7,7"

  line5="沙,干枯,已,揉搓,水,不,来,水,清,过滤,做,渣,不,拥有"
  title="图6"
>}}

再怎么揉搓干燥的砂土，也不会有一滴水从里面渗出来。
再怎么过滤清澈的水，也不会过滤出一点渣滓（原文图6）。

从纯粹到极限的东西中，无论怎样都无法提取出其他的成分。

这句格言中每句话都是七个字，可以按照前四个字+后三个字的结构进行拆分。
每句中的前四个字都是名词句+动词句，后三个字都是名词+“不”+动词的结构。

开头的名词句，都是“名词+形容词”的顺序，也就是类似“砂土←干”、“水←清澈”的结构。
“砂”的字形（𘔏），是由“𘔚”（红）的偏与“𗿀”（土）的旁构成的，表示“红土”，也就是“砂”的意思。
“清澈”的字形（𘄎），是由“𘄂”（纯粹）的偏与“𗨚”（眼）的旁构成的，通过“纯粹的眼”的含义来表达“清澈”的意思。

而后面的动词句，在第一句中是接头词+动词的形式，在第二句中则是动词+助动词（做...）的结构。

第一句中的接头词读作“吃”，表示“朝着说话人的方向进行动作”，是一个指示方向的词。
而翻译为“挤压”的文字，原本是“揉搓”的含义，但在这个语境中，译为“挤压”更加合适。
与之对应的，在第二句中，两个字的含义分别是“过滤”和“做”，连在一起就是“把...过滤”的意思。
以“过滤”的字形为基础，还可以产生很多的派生字：

1. “𘟈”（拔出）：加上“手”作为旁；
2. “𘟉”（蒸汽）：加上“𘔺”（气）的中间与旁；
3. “𗡐𘅴”（疾速）：第一个字是加上“口”作为偏，第二个字是加上“斜”作为偏。表示快速通过某个地方过来的意思。
4. “𗥞𘖗”（救济）中的第二个字，表示“抽出”。

此外，有这样一组词：

1. “𗕰”（读音：叻，，平声十四韵）：拔出，自动词；
2. “𗂍”（读音：縲，，上声十二韵）：拔出，他动词。

这里对立的自/他动词，也都是由前面的“过滤”为基础产生的派生字。

更有意思的是“渣滓”的字形。
在“水”偏加上“重”组成的“沉”字的基础上（见本书第三章第三节第三部分），对旁进行修改，在“重”的上下两个部分之间插入“木”冠，就得到了“渣滓”。
在沉淀的物品中，有“木质”的东西，这就是“渣滓”。

在十一月、腊月借来的衣服，不能随便乱穿”

虽然二月、三月都是贮藏的食物比较少的时期，但还是需要尽量吃靠自己的努力获得的食物，而不是依赖从别人那里借食物。
十一月、十二月都是非常严寒的时期，但还是需要努力，穿靠自己的力量获得的衣物（原文图7）。

这句话每句都是八个字，两句的结构相同，都可以拆分成两个四字片段。
前面的四字是名词句，而后面四字则是名词句（动词+名词）+“不”+动词的结构。

“借用”一词，在前后两句中使用了不同的文字。
“𗔓”表示的是借用食物、钱财那样，并不归还借的东西本身，而是归还具有同等价值的物品。
而第二句中的“𗟫”，则是像借用车或衣物那样，归还借用的物品本身的“借用”。

“𘋳”（衣物）与“𗔛”（穿着）、“𘔋”（债务）与“𗔓”（借用），
两组文字是通过同样的手段派生得到的，
动词是由对应的名词通过加上“𘡊”冠来构成的。

“𗠇”（食物）与“𗠰”（吃喝）这组文字，并没有采用上面那样的派生方法，两者的共同点就只有包含“口”偏这一点。
“𗠰”的含义十分广泛，同时包含了“喝”和“吃”两类含义。

在中国少数民族的语言中，既有将“吃”根据对象进行细分，用不同文字表达的语言，也有像这样用一个词涵盖表达的情况。

## 例6

“平淡的白天不叫白天，骏马疾驰的白天才是真正的白天；
寻常的夜晚不叫夜晚，媳妇进门的夜晚才是真正的夜晚。”

即使是白天，也并非一直都是煞风景的闲暇的状态。
有马儿来回奔跑的忙碌的白天才是理想的。
夜晚也是一样，并不是一直都是孤寂的状态。
有新娘到来给予安慰的夜晚才是令人高兴的（原文图8）。

这两个八言句子，也可分为两组四言句来理解。
第一句和第二句都是“（名词1）都不（名词1），（名词2）（动词）（名词1）好”的结构。

“𗾞”（昼）与“𗾔”（太阳）相关，
而“𘈚”（夜）与“𘈝”（暗夜，包含了“黑”的中间部分与旁）、“𘈞”（梦，包含了“看见”的省略形式）等字相关。

此外，像前面所说的那样，从“皆”字可以派生出“悉”、“一切”等文字，
而“驰”也是在“驰”的基础上加上“马”偏得到的：

- “𘆳”（读音“妹”，mbıěɦ）：驰
- “𘉝”（读音“那”，naɦ）：驰（字形与数字“八”十分相似）
- “𗳵”（读音“驰”，sʷɨn）：驱

“𗃆”（新娘）是以“𘞼”（母）作为音符，产生的一个派生字。
与“𗡸”（妇女）、“𗴺𘖖”（母亲）、“𘞾”（才女）、“𘞽”（女的部族姓氏）等字可以归为一组。
