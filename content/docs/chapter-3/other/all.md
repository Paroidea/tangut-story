# 3. 皆与悉

{{% columns class="book-columns" %}}

- {{< image src="images/chapter-3/all-jie-seal.svg" title="汉字小篆“皆”" class="char" usefigure="yes" >}}

- {{< image src="images/chapter-3/all-xi-seal.svg" title="汉字小篆“悉”" class="char" usefigure="yes" >}}

{{% /columns %}}

汉字的“皆”{{< image src="images/chapter-3/all-jie-seal.svg" title="汉字小篆“皆”" class="char-small" >}}是会意字，由“白”与“比”构成。
也就是说，表示“大家都相同”就是“皆”之意。

“悉”{{< image src="images/chapter-3/all-xi-seal.svg" title="汉字小篆“悉”" class="char-small" >}}，是由“釆”和“心”组成的会意字。
详细辨别事物的区别并列举出来，因此表示“全部”的意思。
两个字合在一起就是“悉皆”，也就是“全部/彻底”的意思。

{{< tangut_highlight 
  line1="𗄊,𗑷"
  line2="皆,悉"
  title="图18 “皆”与“悉”"
>}}

西夏文字中“皆”“悉”的字形来源虽不明，但主要区别在于其冠的不同。
似乎经常以“悉皆”的形式使用（图18）。

{{< tangut_highlight 
  line1="𗄑𗄑,𗄊→𗄭"
  line2="一切,皆→集"
  title="图19 经典中常用的“一切”"
>}}

在经典中常用的“一切”，就是由该字形派生的，两个字叠用作为副词使用（图19）。
“一切”的字形，是把“皆”的旁替换为“聚集”派生出来的。
把“全部”给“聚集”起来，自然而然就是“一切”的意思。

{{< tangut_highlight 
  line1="𗩾𗨁"
  line2="最上"
  title="图20 “最上”"
>}}

从“皆”派生出来的字形中，还有一个重要的字是“最”。
这个字是在“皆”的基础上加上“人”偏得到的，但发音与“皆”完全相同。
例如“最上/至上”（图20），在字形上虽然用的是“最”，但在发音上，与“皆之上”相同。
