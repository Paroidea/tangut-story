汉字的“上”和“下”，如众所周知，是以某个基准来表示其上与下的指事字。

{{< tangut_highlight 
  line1="𗹭,𗤢"
  line2="上,下"
  title="图9 “上”与“下”"
>}}

在西夏文字中，“上”字，是由类似汉字“肩”的字形而构成；
“下”则是把“人”偏与“梯子”组合而成的字形（图9）。

{{< tangut_highlight 
  line1="𘏈,𘓤,𗊢→𘓥,𗴿𗒀"
  line2="肩,背负,重→货物,供奉"
  title="图10 包含“上”作为偏旁的字"
>}}

“上”是单体字，有很多从其派生出来的字。
首先，加上“手”偏，就成为“手的上部”，即“肩”的意思。
又如将“荷”的旁作为偏旁，组合而成的文字就是“背负”的含义。
顺带一提，“行李”大体上都是比较重的东西，因此“荷”字也包含了“重”字中间的部件。
作为“供奉”之意的词，在经典中也常使用。
“供奉”的“奉”字，也使用了这个“上”字作为旁。
这个字形，表示被供奉的是上方的事物（图10）。

{{< tangut_highlight 
  line1="𗣛,𘃰,𘋳,𘘛𘀒,𘚈,𗴺"
  line2="高,襟,服,北斗,祖,母"
  title="图11 包含“上”的文字"
>}}

此外，包含“上”的文字很多，其中一个比较有趣的是加上“人”偏形成的字，表示汉人的姓“高”。

把“衣服”的中央部分作为偏，就表示衣服的上部，即“襟”。

把“教/学”作为偏，形成的文字就是“北斗星”里的第一个文字。
表示“因为更高所以可以教导别人”的意思。

而“祖/先”，则是“母”的简略形式与“高”的组合而成的字（图11）。

{{< tangut_highlight 
  line1="𗤢→𗩞,𗩞𗹭"
  line2="低→低,高低"
  title="图12 包含“下”的文字"
>}}

{{< tangut_highlight 
  line1="𗝕𗥋,𗭹𘄩"
  line2="吹笛,击鼓"
  title="西夏语小单词集 - 24"
>}}

“下”字的旁是一个单体字，单独使用时表示“梯子”的意思。
加上人偏之后，不知为何成为“下”的意思，有时也表示“低”的意义，有着非常多的派生字。

在“下”字的右边加上偏，就表示“低”的意思。
在这里，这个偏起着音符的作用，
单独成字时读音为上声十韵，加上偏之后则是平声十一韵。
虽然发音里都包含“迷”的元素，但声调不同。
声调的差别可以通过这个旁的有无被体现。
这一字形，与发音完全相同的“光”字（𘉍，音“迷”，平声十一韵）具有相同的旁。

此外，这个字还可以与“上”一起使用，表示“高低”的含义（图12）。

{{< tangut_highlight 
  line1="𗄽"
  title="图13 山"
>}}

对“低”进行否定，表示“不低的地方”，就成为了“山”。
如前面所述，西夏语固有的“山”字还有另外两个，这个字是专为书写汉语中的借用词“山”而用的（图13）。

对于“上下”的概念，也可以将前面的“上”/“下”作为第二音节，通过一个双音节词整体来表现（图15）。

两个词中的第一个文字都包含“口”偏，读音完全相同（平声三十韵，“迷”）。
这个是表示某种基准的语言，如果强行地逐字翻译的话就表示“高度在上”、“高度在下”的意思。
这个文字包含“口”偏的原因，大概是因为西夏人在判断上（高）或下（低）的时候，基本都是以自己的嘴的高低为基准进行判断的。

{{< tangut_highlight 
  line1="𗳲𗳇𗹭,𘜳𗱒"
  line2="丘陵上,洮口"
  title="图16 包含“上”的地名"
>}}

到目前为止，已经发现了多个像图16里列举的这样的包含“上”的地名。
但是还不知道它们对应的具体位置。

“洮口”中最后一个文字“𗱒”，除了在地名中使用以外，还表示“桃色”的意思。
根据西夏人的注解，它表示“𘜳𘄀”即汉语中的“桃朱”（即粉色）。
