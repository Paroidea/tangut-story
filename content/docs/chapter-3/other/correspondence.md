# 4. 西夏语与汉语的对译关系

{{< tangut_highlight

这里出现了一个问题。
关于“最上”，如果把西夏字分开来看，把那两个字对应到汉字的“最上”是最合适的。
但在像图21这样的语境中，把它翻译成“最乐世界”则是不合适的。
反而把它理解为“极乐世界”才是正确的。
此外，使用了同样文字的“𗩾𗥃”，翻译成“最妙”、“极妙”都是合适的。
这个词还可以对应到“微妙”。
另外，还有“𗩾𘓋”，可以翻译为“大略”。

类似这样，同即便是同一个西夏字，也并非总是与同一个汉字相对应。
而是需要根据上下文，或是单词的不同而发生变化。
在各自的语境下，根据上下文确定最为恰当译文的一项重要工作。

碰到与汉语的对照关系存在问题时，需要经典文献等资料，检查原文中汉语是如何被翻译的。
首先需要从西夏文字的原义（字义）出发。
例如，图23中的词语“金铸像”，可以对比为(a)是“金”，(b)是“铸”，(c)是“像”的关系。
但(b)在实际上并不是“铸”的意思，而是表示“散/散布”。
也就是说，这里把汉语“金铸像”翻译成了“表面撒有金的像”的意思。

再举一个例子，图24是汉语中一种云的形态“段云”的译语。
如果直接对应的话，前面的字是“云”，后面的字就是“段”。
但实际上，后面的字实际上是“结绳”的意思，也就是翻译成了“如同把绳子打结那样状态的云”。

类似这样，汉语与西夏语的对译关系，虽然看起来可以较为简单地对照，但实际上问题往往相当复杂。
