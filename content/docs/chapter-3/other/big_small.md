# 1. 大与小

{{% columns class="book-columns" %}}

- {{< image src="images/chapter-3/big-seal.svg" title="汉字小篆“大”" class="char" usefigure="yes" >}}

- {{< image src="images/chapter-3/small-oracle.svg" title="汉字甲骨文“小”" class="char" usefigure="yes" >}}

{{% /columns %}}

汉字的“大”，是从正面看的“人的手脚张开”以表示“大”的指事字。
而“小”，则是以三个小点表示物体的微小，也是指事字。

{{< tangut_highlight

>}}

西夏字中，虽然“大”与“小”字都可以拆分为偏和旁来进行分析（图1），但“大”字的偏，并不一定总是表示“大”的意义，在单独使用时，它表示的是“凉快”（图2）。

{{< tangut_highlight 
  line1="𘊡𗿳,𘜸,𘓷"
  line2="季节,寺院,身体"
  title="图3 包含“大”的字"
>}}

此外，在包含“大”字的偏的文字中，还有“季节”之“季”字（图3）。
一年中不同季节有很大的差别，因此与“大”有关。
最后，在“寺院”或“身体”等字之中，不知为何也包含“大”这一要素。

{{< tangut_highlight 
  line1="𘈩,→,𘛽"
  line2="一,→,身体"
  title="图4 身体"
>}}

西夏语中一般使用的“身体”，是把数词“一”作为音符构成的字形，其含义反而更接近汉语的“躯”（图4）。

{{< tangut_highlight 
  line1="𘜴,𗿒"
  line2="大,大、太"
  title="图5 其他的“大”字"
>}}

像图5中所示的两个字一样，“大”字还有其他的写法。

其中的第二个字形，是把前面提到的“大”字左右要素置换而成的对称字，大概是从汉语借用而来的词汇。

{{< tangut_highlight 
  line1="𘜷𗗚,𘜺𗗚"
  line2="南海,北海"
  title="图6 其他的使用了“大”的字"
>}}

“南海”、“北海”里面的“南”、“北”，也用到了这个“大”的偏。
或许其中暗示了“大海”的含义（图6）。

西夏文字中的“小”，是由“人”偏与“小”构成。
虽然不知道为什么是这样的组成，但这个字同时有“小”和“亦”两种含义。
而以这个表示“小”的偏旁作为意符产生的文字并不少见（图7）。
