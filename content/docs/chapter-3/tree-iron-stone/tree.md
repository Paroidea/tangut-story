# 1. 木与树


{{< image src="images/chapter-3/wood-seal.svg" title="汉字小篆“木”" class="char" usefigure="yes" >}}

{{< image src="images/chapter-3/tree-seal.svg" title="汉字小篆“树”" class="char" usefigure="yes" >}}

汉字的“木”（{{< image src="images/chapter-3/wood-seal.svg" title="汉字小篆“木”" class="char-small" >}}），是象征生长着树枝的树干的象形字。
而“树”（樹），则是由“尌”（读音：术）作为音符的形声字。

{{< tangut_highlight
    line1="𘕰,𗝠" 
    line2="木,树" 
    title="图1 “木”与“树”"
>}}

而在西夏字中，“木”对应的是“𘕰”（读音：普），而“树”则是“𗝠”（读音：西）。

{{< tangut_highlight
    line1="𘕵,𗚋,𘞵,𘞶𘕷" 
    line2="(a) 树梢,(b) 小枝,(c) 长大,(d) 枝叶繁茂的样子" 
    title="图2 “木”的派生字"
>}}

表示“木”的“𘕰”是一个单体字，可以单独使用。
加上“指”部形成的派生字(a)，表示“树枝”的意思。
“木”冠配上“指”形成的字是(b)“小枝”的意思。
此外，把表示(c)“长大”含义的文字的旁替换成“木”，形成的对称字(d)，表示“枝叶繁茂的样子”（图2）。

另一方面，表示“树”的“𗝠”字，则是“木”冠、“人”偏以及表示“聚集”的“𗢬”所构成的。
就像表示各种树木的名称的文字，在汉字中都带有“木”字旁一样，
西夏字中，这类文字大部分都带有“木”冠。

{{< tangut_highlight
    line1="𗞓,𗟄,𗞌,𗛕,𗚵,𗚐,𗝹,𘚐,𗞠,𗛁,𗚄" 
    line2="樱,松,柏,桂,柳,杨,榆,桑,杏,栗,桃" 
    title="图3 表示“树木”的代表性文字"
>}}

西夏人编纂的单词集《（三才）杂字》中，专门有一项是树木相关的文字，其中就有很多带有“木”冠的文字。
但具体这些文字分别指的是哪一种树，现在还有很多是无法解读的。
图3中列举了一些目前能够判明的代表性的树名。

其中大部分的文字，都是将“木”冠下的原字作为音符，配上“木”冠之后形成的字形，读音与原字保持一致。

“𗜋𗝠”（海棠树），是在汉语中“海”的含义对应的文字“𗗚”的基础上，配上“木”冠形成的文字。
类似这样的例子还有很多，并不少见。

{{< tangut_highlight
    line1="𗜗𗟒,𗞈𗚄" 
    line2="龙柏花,葡萄" 
    title="图4 包含木冠的表示花/国务的文字"
>}}

由于西夏文字中，并没有区别于“木”冠的专门用于花相关的“花”偏或是“花”冠，因此花与果实相关的文字，也有一些是带有“木”冠的文字（图4）。

“龙柏”,“葡萄”这两个词，都是汉语中对应文字的音译。
而在字形上则是由“木”冠组成的形声字。
“葡萄”一词中的第二个字，原本是“桃”的意思。

虽然对野菜相关的文字来说，由“菜”偏可以使用，但“芜荑”这类词仍然是由汉语音译的，通过“木”冠组成的字形来表示（图5）。

{{< tangut_highlight
    line1="𗜨𗟑,𗛆,𗜱,𗚶,𗛄,𗝊,𗛶,𗚯,𗞳𗟎"

    line3=",,木冠+前端,木冠+细,,,,," 
    title="图5 包含木冠的文字群"
>}}

表示树木上不同部位的文字，也由“木”冠组成，这些文字大多是会意字。

“𗝻𗛼”（树瘤）中，两个字分别是“𗋢”（绳）和“𗲄”（结）的派生字。
其中第一个字加上了“木”冠，第二个字则是把“结”的冠替换成“木”冠形成的文字。

建筑物不同部位的文字也大多具有“木”冠，这些文字中很多都是形声字。

“𗞁”（柱）字是由“𘚪”（立）作为音符的派生文字，是一个会意字。

木制的道具对应的文字，很多也都具有木冠。

然而，像是“契丹”、“出现/显现”、“杀”这样的文字，根本难以理解为何其中会包含“木”冠。这样的例子也并不少见。
