# 2. 铁

{{< image src="images/chapter-3/iron-seal.svg" title="汉字小篆“铁”" class="char" usefigure="yes" >}}

汉字中的“铁”（鐵）写作 {{< image src="images/chapter-3/iron-seal.svg" title="汉字小篆“铁”" class="char-small" >}}[^tie]，是“金”偏加上音符“𢧜”形成的形声字。
由于铁具有“黑色金属”的意思，因此也有人认为“𢧜”是音符。

{{< tangut_highlight
    line1="𘟪" 
    title="图7 “铁”"
>}}

西夏字中的“铁”的字形构成虽然没有办法理解，但它确实是一个由“金”冠组成的基本文字。
除去“金”冠以外的部分呈现出对称的结构，不会被单独使用（图7）。

{{< tangut_highlight
    line1="𘟬,𗍢𘃐" 
    line2="斧,锹" 
    title="图8 “铁”的派生字"
>}}

从铁出发可以产生“战斗”以及“武器”等派生字。
除此之外，包括铁的字型在内，还有数个通过添接法产生的文字。

其中一个例子就是“斧”以及“锹”。

锹是以“壁”（𗌷）作为音符，旁替换为“铁”产生的文字。

{{< tangut_highlight
    line1="𘗊,𘗅,𘖾,𘗁,𘟞,𘖵,𘗋,𘗎,𘖰" 
    line2="铜,锡,鍮(黄铜),钢,刀,锯,剪刀,釜,锅" 
    line3="金冠+红,金冠+柔软,金冠+制作,金冠+优越,金冠+小,金冠+切成两半,金冠+断,金冠+沸腾,金冠+温暖" 
    title="图9 “金”冠的会意字"
>}}

“金”冠组成的文字，首先是表示各种金属的文字。

汉字中表示金属的文字全部都是包含音符的形声字，
与之对应的，西夏字中，这些都是会意字，
而且都是由表示各种金属的代表性的特征的文字构成的。
因此在字形的层面上，铜是“红”，锡是“柔软”，钢是“优越”，可以进行类似这样的分析。
这是很有意思的一个现象（图9）。

而对于金属制的道具来说，像前面提到的带有“金”冠和“细小”所组成的“针”这样的文字一样，大多也都具有“金”冠。

{{< tangut_highlight 
  line1="𗋽𗕉,𘁑𗄋,𘀓𗺱,𗿼𘎃"
  line2="川獭,猫,鼠,飞禽"
  title="西夏语小单词集 - 12"
>}}

像前面所说的那样，包含“金”偏“钅”的汉字所对应的西夏字，并非全部都带有金冠。

比如说铃铛的“铃”与“铎”在汉字中，因为它的构成材料是金属，因此具有“金”偏。
而在西夏文字中，倒也确实有带有金冠的铃字和铎字。

{{< image src="images/chapter-3/section-3_fig-10.svg" title="图10 包含“吵闹”的“金”冠文字" usefigure="yes" >}}

> 译者注：原书中图10第二行的两个文字，中间部分均为“𘢩”。

然而“铃”和“铎”两者都有“能够发出声音，比较吵闹”的性质，
因此是通过加上“金”冠（铃）或者“𘡊”冠（铎）的形式被派生出来的。
两者都是会意字。
在西夏人的认识中，不管是“铃”还是“铎”，最为明显的特征就是他们都是比较“吵闹”的存在。

而进一步回溯去考虑这个“吵闹”的字形，我们发现它是在“鸣响、雷鸣”的基础上，不知为何添加了“水”偏而得到的文字。
除此之外，他也有加上人偏得到的字形，这个也是“吵闹”的意思。

{{< tangut_highlight
    line1="𗐴,→,𗒨" 
    line2="耳,→,钹" 
    title="图11 文字层面上表达“吵闹”的字"
>}}

类似这样，西夏人对能发出声音的东西，都具有非常有趣的理解方式。

“铙”和“钹”也都是非常吵闹的乐器。

在汉字中“钹”是由“金”偏和音符“犮”所组成的形声字，但西夏人则是将“耳”创造出来的派生字作为“钹”。
当“钹”在耳朵边上发出声音时，耳朵会感觉非常吵闹。
这一事实也在文字层面上得到的表现（图11）。

  line1="𗿻𘜾,𗀝𗀪,𘟦𗀝,𘎃𗾢,𘂕𗾢,𗴍𘔿"
  line2="凤凰,孔雀,鸳鸯,雀,燕,鹌鹑"
  title="西夏语小单词集 - 13"

除了上面提到的“铃”字以外，还有另一个“铃”字，它是由“音”偏所构成的。
“钟”和“法螺贝”也同样都具有音偏。
由于它们都是可以发出声音的物体，因此包含“音”偏是理所当然的。
“音”偏构成的西夏字的数量也很多，比如“寂静”是由“音”偏加上“无”所构成的，
也就是“没有声音”的意思，自然而然就是“安静”了（图12）。

{{< tangut_highlight
    line1="𗙗,𗙘,𗗚𗘾,𗘺,𗙧,𗙫" 
    line2="钟,铃,法螺（逐字翻译为海螺）,寂静,繁荣,唵（咒文用语）" 
    title="图12 “音”偏的文字"
>}}

“繁荣”也是由“音”偏所构成的字形，恐怕表示的是“名声传播地很广”所代表的状态。
西夏文的咒文中，“唵嘛呢叭咪吽”是在以敦煌莫高窟为首的各种地方被发现的，其第一个文字“唵”也是由“音”偏所构成的。
“音”偏加上“诏”字得到的就是这个文字。

西夏文字的结构，能够表现出西夏人各种各样不同的想法。
