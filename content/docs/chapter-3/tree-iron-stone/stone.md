# 3. 石与重

{{% columns class="book-columns" %}}

- {{< image src="images/chapter-3/stone-seal.svg" title="汉字小篆“石”" class="char" usefigure="yes" >}}

- {{< image src="images/chapter-3/heavy-seal.svg" title="汉字小篆“重”" class="char" usefigure="yes" >}}

{{% /columns %}}

汉字里的“石”和“重”，在字形上并没有联系。

“石”字{{< image src="images/chapter-3/stone-seal.svg" title="汉字小篆“石”" class="char-small"  >}}是石块和音符组合而成的形声字，另一方面，“重”则是“人”与“拿着货物的样子”组合而成的会意字 {{< image src="images/chapter-3/heavy-seal.svg" title="汉字小篆“重”" class="char-small" >}}。

>}}

而在西夏字中，“石”和“重”在字形上有所关联（图13）。
石头被认为是重的东西，两者都包含有“石”冠“𘢸”。
“石”的字形，或许山有所关联，
而“重”字则是将两个“石”冠重叠在一起形成的字形（下方的“石”冠字形有少许的修正）。

{{< tangut_highlight 
  line1="𗾈𘓐,𗭩𘓐,𗤶𗣋"
  line2="贤人,愚人,心命"
  title="西夏语小单词集 - 15"
>}}

从这个“重”字产生的派生字的数量非常之多（图14）：
1. 以加上"水"偏，就变成了下沉的“沉”字。
这是因为水中的重物，全部都会下沉。
而“水中沉下的东西留存下来”就对应着“渣滓”，这个字也是在“沉”字的旁里加上“木”冠所得到的排生字。
而直接在“沉”的顶上加上“木”冠，就得到了“沉香”的“沉”字。
2. 加上“手”偏，就得到了投掷的“掷”。对应着“每个人都可以把手上拿着的重物投掷出去”。
3. 加上“人”偏，那么就表示“病情很重”的意思。
4. 加上“土”偏，表示地上很重，也就是水都沉到地里去的样子，也就是汉字里干涸的“涸”字。

{{< image src="images/chapter-3/section-3_fig-14.svg" title="图14 “重”的派生字" usefigure="yes" >}}

与字形“重”相关联的，还有一个对称文字“平等”。
中间的一竖分隔的左右两边，都是“重”，分布非常均匀，因此就是“均衡”的状态。

只留存一边的话，那么就表示“分配重量使之均衡”的含义，也就是“均分”的意思。
在“均”字旁边加上“手”偏，就表示用手来使之均衡，也就是“秤”的含义。

在“均”字旁边加上“眼”偏，那么就表示“以平等的目光看待事物的人”，也就是“长官”的含义。这个字也在“县丞”（县里的长官）等词语中被使用。
