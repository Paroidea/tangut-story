# 10. 咽喉

{{< image src="images/chapter-3/throat-yan-seal.gif" title="汉字小篆“咽”" class="char" usefigure="yes" >}}

{{< image src="images/chapter-3/throat-hou-seal.gif" title="汉字小篆“喉”" class="char" usefigure="yes" >}}

汉字的“咽”和“喉”都是形声字，是由口字旁加上音符“因”或者“侯”组成的。

{{< tangut_highlight 
  line1="𘙴,𗍂,𗠛"
  line2="咽,喉,吞咽/谋生"
  title="图31 “咽”、“喉”、以及“谋生”"
>}}

而西夏文字的“咽”，是“长”偏加上“口”组成，“喉”则是与“吞咽、谋生”对称的字形。
喉咙在每个人身上都是一个长度较长的器官，对人类的生命来说至关重要，因此在西夏字中具有这样的字形。
在西夏人独特的表现方式中，咽喉被认为是饮食生活的基准（图31）。

“长”是西夏字中的基本字之一，从它派生出了很多的文字（图32~39）。

  line1="𘙲,→,𗓌,𘐒,𗁅𗅌𘐒,𘑇,𘙸,𗽿,→,𘙼"

  cspan3="1,1,1,1,1,1,1,3"

在顶上加上冠作为动词使用，就表示“变长”的意思。
咒文（陀罗尼）中会把这个字，用在“把读音拉长”的指令中。

图34的“屈”字是“直”字进行否定得到的字形。
在“伸”的旁边加上“𘠴”旁，得到的文字表示“张开”的意思。
另外，在“长”的对称字中间加一竖，就是“串”的意思。
这个字形表示了把长长的物体串起来的状态。

把“长”作为偏，再加上“尖”字的冠与旁，就得到了“嘴”字。
嘴大多都具有较尖较长的形态，因此这个字形比较容易记忆。

  line1="𘙶,𘙲,→,𗟥,→,𘙳,𘖑,𗫡,→,𘖎"

把“长”的偏配上文字“狭窄”，就表示“又长又狭窄的物体”，也就是“条”。

把“长”的偏配上文字“除”的偏，表示“把长的东西除去”，也就是“缩短”。

顺便一提，“短”字是将“尚未”作为偏，“抵达”作为旁组成的字。
因为“还没有抵达”所以是“短”的（图40）。
