# 6. 牙齿

{{< image src="images/chapter-3/tooth-chi-oracle.svg" title="汉字甲骨文“齿”" class="char" usefigure="yes" >}}

{{< image src="images/chapter-3/tooth-chi-seal.svg" title="汉字小篆“齿”" class="char" usefigure="yes" >}}

汉字中的“齿”（小篆：{{< image src="images/chapter-3/tooth-chi-seal.svg" title="汉字小篆“齿”" class="char-small" >}}），原本在甲骨文中的字形是“{{< image src="images/chapter-3/tooth-chi-oracle.svg" title="汉字甲骨文“齿”" class="char-small" >}}”。
这一字形来源于牙齿从前面看的时候呈现的样子，是一个象形字。
随后它被加上了音符“止”，成为了我们今天所看到的字形。

{{< image src="images/chapter-3/tooth-seal.svg" title="汉字小篆“牙”" class="char" usefigure="yes" >}}

“牙”也是一个象形字，小篆中“{{< image src="images/chapter-3/tooth-seal.svg" title="汉字小篆“牙”" class="char-small" >}}”的字形表现了野兽的巨大牙齿咬合时候的样子。

{{< tangut_highlight 
  line1="𘟗,𘘄"
  line2="牙,齿"
  title="图14 “牙”与“齿”"
>}}

而在西夏文字中，究竟为什么“牙”与“齿”会拥有图14这样的字形，现在完全无法搞清楚。
但这两个字形中总感觉有一点汉字里“牙”与“齿”的感觉。

{{< tangut_highlight 
  line1="𘘅,𘘃,𗁏,𗉩"
  line2="咬（咀嚼）,咬（撕咬）"
  cspan2="2,2"
  title="图15 “齿”的派生字"
>}}

从“齿”产生的派生字中（图15），有好几个都具有“咬合”或“撕咬”的含义。
但不同文字的含义之间的细微差别现在还没能理解透彻。

{{< tangut_highlight 
  line1="𘖷"
  line2="锯齿"
>}}

把“齿”的笔画数减少，然后再加上金冠形成的字形，或许表示“锯齿”的含义。

{{< tangut_highlight 
  line1="𘘇,𗛘"
  line2="狼,豺"
  title="图16 包含部首“齿”的动物"
>}}

同时，西夏人认为狼的特征在于其牙齿，因此“狼”的字形是根据“齿”与“兽”这两个字创造出来的。

{{< tangut_highlight 
  line1="𘟘"
  line2="咀嚼"
>}}

从“牙”字，也产生出了表示“咀嚼”的派生字。

{{< tangut_highlight 
  line1="𗅷𗆝,→,𘒗,𘒗𗼺"
  line2="(a) 牙齿,→,(b) 门牙,(c) 齿植（长着牙齿的骆驼）"
  title="图17 “齿”的另一字形及其派生字"
>}}

实际上，除了前面提到的西夏字以外，“牙齿”还有另一种写法（图17）。
但这一写法与前面的“牙”与“齿”究竟有什么区别现在还不清楚。

在这一种“牙齿”的写法的基础上，加上一个冠，就可以派生出文字(b)（表示门牙的意思）。
这个字可以作为(c)“骆驼”的另一种叫法来使用；我觉得也有可能是表示牙齿排列在一起的意思，但目前还没有确切的证据。
