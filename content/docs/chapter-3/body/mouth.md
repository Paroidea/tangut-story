# 3. 口

{{< image src="images/chapter-3/mouth-seal.svg" title="汉字小篆“口”" class="char" usefigure="yes" >}}

汉字里的“口”，从字形上就能非常容易地看出来，是仿照人类的嘴的形态创造的简单的字形。
而西夏文字的“口”（图7），则具有比较复杂的字形，是由人偏和“口”部首组成的会意字，
发音为“烈”（hlǐah，上声十八韵）。

{{< tangut_highlight
    line1="𗢸" 
    title="图7 口"
>}}

包含部首“口”的文字数量很多（图8）。
把两个“口”分别放在偏和旁的位置排列在一起，就形成了“唇”；
再把偏替换成“水”，就意味着(b)口里的水，也就是唾液的含义。
把口偏和水旁写在一起的字形(c)，恐怕也与(b)一样，对应“唾液”的意思。

{{< image src="images/chapter-3/section-2_fig-8.svg" title="图8 “口”的派生字" usefigure="yes" >}}

> 译者注：原书中“吕”字的旁为“𘠆”，但《古今文字集成》和各Unicode字体中用的都是“𘠢”。

“唇”的偏与旁之间加上“水”，形成的字形(d)表示口中流动的水，也就是口水的意思。
它的读法是“赎”（tšhǐu，上声二韵）。

还有一种非常有趣的造字方法，就是将“口”上下排列在一起，再配上一个意义不明的旁(e)，就形成了(f)的字形。
这个字是汉人的姓“吕”的含义。
也就是说，西夏字“吕”也仿照汉字“吕”，是用两个“口”上下重叠来组成的。

《孝经》的原本已经散失，有一版古老的注释本，是吕惠卿的注释本。
其西夏语译本中的前18章流传了下列，现在藏于列宁格勒东方学研究所中。
但其实物是用草书体书写的，内容判别起来非常困难。

伦敦的斯坦因的收藏品中，也有楷书体抄写的一本《孝经》。
我所找到的是在书的序的结尾部分的内容，其中当然理应会有“吕”字存在。
然而不幸的是，那一部分的内容破损了，只有“惠卿”两字还留存下来。

把表示(g)“水枯竭了”的文字加上口偏，就得到了派生字(h)“口渴”，这是一个典型的会意字。
(g)的读音是“”（saɦ，上声十四韵），而(h)的读音是“”（pa，上声五十六韵）。

在(j)字形的基础上加上口偏，就得到了(i)“喝”的文字，发音为“”（thiɦ，平声十一韵）。
佛典中经常出现的“醍醐”一词，指的是“味道最好的好喝的酒”，引申义也可以指佛法的妙理。
在西夏文翻译中，通常都音译为(k)的形式。
原本“醍”对应的字指的是“喝”的意思，而“醐”并没有特别的含义，只是根据其发音写成(l)的形式。
这样一来，根据其“喝起来很好喝”的含义，将汉语中“醍醐”一词通过“喝”的文字进行了翻译。
即便是在当今社会，使用汉字对外来语进行音译的操作仍然能经常看到。
