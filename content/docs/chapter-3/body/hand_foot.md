# 8. 手与足

{{< image src="images/chapter-3/hand-bronze.svg" title="汉字金文“手”" class="char" usefigure="yes" >}}

{{< image src="images/chapter-3/stop-seal.svg" title="汉字小篆“止”（足）" class="char" usefigure="yes" >}}

汉字的“手”与“足”是象形字，象征着人类的手（五根手指展开的样子）与足（原本的字形是“止”，脚踝以下部分的形状）。
因此，“手”与“足”的字形之间没有相互关联。
但在西夏字中，由于“手”与“足”都表示身体的中轴以外伸出来的部分，因此包含有一些相同的要素。
两者的旁是完全相同的，只有偏存在差别（图20）。

西夏字中的“手”与“足”也是比较基本的文字，从它们出发产生的派生字的数量并不少。

下面我列举一些代表性的文字。

“束缚”的西夏字，是将“手”与“足”的偏替换为“连接”含义的部首形成的。
原本表示“手、足连在一起”的含义。
将“手”与“足”的旁分别改变成“束缚”的偏，的大的文字分别表示“手铐”、“脚铐”的含义。

此外，“束缚”的对称字是“连接”的意思，在它左边加上一竖得到的文字，表示“接受、被袭击”的意思（图26）。

{{< tangut_highlight 
  line1="𘃍,𘃎,𗀺"

  title="图27 “指”的派生字"
>}}

同时，与“手”的字形相近的还有一个字“指”，其对称字表示“指甲”的意思。
“掌”字也是其派生字之一。

“掩住耳朵”的“掩”字用的也是这个“掌”字，表现了“用手掌掩住耳朵”的状态。
