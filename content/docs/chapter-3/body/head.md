# 1. 头

汉字中的“头”（頭），是象形表示人低头行稽首之礼的动作的“頁”和和音符“豆”所组成的。

{{< tangut_highlight
    line1="𗥦" 
    line2=" 头 "
    title="图1 西夏文字：头"
>}}

而在西夏文字中（图1），“头”可以看作是由人偏、中央部分（头）和右边的旁（含义不明）所组成的基本字。首先，将右边的旁替换成表示“先前、过去”的旁，就形成了表示“开始”含义的文字（图2a）。

{{< image src="images/chapter-3/section-2_fig-2.svg" title="图2 “头”的派生字群" usefigure="yes" >}}

接下来，将“开始”的中间部分，换成表示“上”的部件，就形成了(b)“先前”。
将“先前”的中间部分再换成前面提到的“年”的旁，就形成了单词(c)“前年”中除去“年”以外的部分。
西夏语中的“前年”是双音节单词，因此用两个字符来表示。

把“头”的旁替换成“𘡧”，就形成了(d)“后天”。这也是一个包含两个字的单词。

除了这些以外，还有一些从“头”创造出来的派生字。
比如，把“头”中右边的旁变化一下，就变成了(e)“发髻”。西夏人似乎把发髻看作是某种混合的事物一样。

把“头”的中间部分替换成“先前”，就成为了(f)“脸”；
把“头”的偏替换进另一个(g)“脸”的偏，就形成了(h)“额头”。
在“脸”的上方就是“额头”，因此“额头”的文字会有这样的组成方式。

把“脸”的旁替换成“𘡭”，表示脸上非常喜悦的样子，就形成了(i)汉字中借用的“庆祝”的文字。

类似这样，很多西夏文字之间都存在相互关系，但首先，我们需要将中心部分的文字记住。
对于刚刚介绍的这个文字网络来说，“头”、“起始”、“额头”、“脸”都是非常重要的文字。

西夏文字中还有另一个字也具有“头”的含义（图3j）。
这个字属于对称字，在含义上与前面介绍的“头”字是完全一样的。
(k)“脑”和(l)“顶头”的文字也属于这个“头”字派生出来的。
其中后者或许将是(m)“元”字作为意符所派生出的文字。

{{< image src="images/chapter-3/section-2_fig-3.svg" title="图3 另一个“头”字的派生字群" usefigure="yes" >}}

进一步地，从它还可以派生出(n)“顶部的毛发”的文字。
将“顶”的旁与(o)“头发”的旁进行置换，就能得到这个字。
(p)“戴上笠”是将“头”的下半部分替换成“人”得到的派生字。
这个字的字形就像在表现“人的头上带上斗笠”这样的感觉一样。

类似构成结构的还有图3(q)中的字，上半部分是“人”，下半部分是“马”。
合在一起就是“骑马”的意思。

西夏文字中，虽然数量不多，但也有像这样具有象形含义的字形存在。
