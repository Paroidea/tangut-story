# 7. 舌头

{{< image src="images/chapter-3/tounge-oracle.svg" title="汉字甲骨文“舌”" class="char" usefigure="yes" >}}

汉字的“舌”（{{< image src="images/chapter-3/tounge-oracle.svg" title="汉字甲骨文“舌”" class="char-small" >}}）是表现舌头从口中伸出的样子的象形字，
但西夏字“舌”（图18），则是人偏与部首“舌”所构成的。

{{< tangut_highlight 
  line1="𗢯"
  title="图18 舌头"
>}}

舌头自古以来就被认为是知晓味道、帮助发音的一个器官。
根据西夏人的注解，舌头是“尝味道的物体、诉说语言的物体”。

{{< tangut_highlight 
  line1="𘀼,𗗘"
  line2="说明,甜味"
>}}

但在西夏人对字形的说明中，将人偏作为“说明”的旁，或是将“舌”作为“甜”的偏的原因，并不是非常简单地就能阐明的内容。

“甜”字在西夏文中，是由“舌”与“倾斜”两部分组成的。
可以解释为因为甜的东西很美味，因此舌头会倾斜过来。

从“舌”产生的派生字（图19）中，有一个字是“𗤰”（酸味），将其进行动词化之后得到的字“𗒴”（酿造醋）也是同一个系统的文字。
将“酿造醋”的偏替换成禾偏，得到的字表示“酒曲”的意思。

{{< image src="images/chapter-3/section-2_fig-19.svg" title="图19 “舌”的派生字" usefigure="yes" >}}

顺便还要提到的是，将“舌”中的人偏替换成水偏，表示“味道淡”的意思。
在“甜”的旁边加上水偏，就表示“甜味”的意思。
在右边加上“虫”，就表示“蜜”的意思。

此外，在“甜”的旁边加上口偏，得到的字形表示“嘴里感觉很好”，也就是“美味”的意思。

类似这样，表示不同味道的字形，基本都是在“舌”的基础上创造出来的。
