# 9. 腹部

{{< image src="images/chapter-3/belly-seal.svg" title="汉字小篆“腹”" class="char" usefigure="yes" >}}

汉字的“腹”是表示“肉”的部首和音符“复”组成的形声字，“肚”也同样是形声字，由音符“土”构成。
腹部从人的外部看起来，是身上最为容易膨胀的部分，而“肚”则表示腹部内部的部分。

{{< tangut_highlight 
  line1="𘀟,𗏎"
  line2="腹,肚"
  title="图28 “腹”与“肚”"
>}}

而西夏字的“腹”，则是由“𘀥”（胃，“腹”偏加上“袋”旁）以及“人”旁所构成的。
根据西夏人的注解，“腹”指的是“储存肠与粪便的囊”，与汉字的“腹”具有相似的含义，我认为恐怕指的是全部的内脏（图28）。

“饿”、“饱”这些字也具有与“腹”字相关联的字形（图29）。

{{< image src="images/chapter-3/section-2_fig-29.svg" title="图29 “腹”的派生字群" usefigure="yes" >}}

而“肚”的西夏字表示的含义是“盘绕、螺旋状”，或许指的是人类的内脏“肠”的具体形态。

而对于内脏“肠”来说，它在西夏语中也有若干个对应的字形（图30）。
