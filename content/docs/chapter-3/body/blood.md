# 11. 血

{{< image src="images/chapter-3/blood-oracle.svg" title="汉字甲骨文“血”" class="char" usefigure="yes" >}}

汉字的“血”的字形，原本象征的是在器皿中盛放的东西，也就是祭品的血液。

虽然“血”与“皿”之间的关联，在现在的文化中早已被忘记，但其仍然通过字形的形式传承到了今天。

{{< tangut_highlight 
  line1="𗊴"
  title="图41 “血”"
>}}

西夏文字中“血”的字形（图41）到底是从何而来，现在已经没有办法搞清了。
但至少与器皿是没有关联的。
“血”字是由“水”偏、“𘤣”、“𘠢”三部分构成的，派生字的数量并不在少数（图42）。

{{< image src="images/chapter-3/section-2_fig-42.svg" title="图42 “血”的派生字" usefigure="yes" >}}

1. 加上表示“聚集”的冠，就表示“脓”的意思，也就是“血在聚集”的含义；
2. 加上“土”偏，就表示“血块”的意思，也就是“像土一样的血的集合”的含义；
3. 加上“皮”偏，表示“血脉、血管”的意思，也就是“让血流过的皮”的含义；

    在汉字中，也有表示类似含义的“衇”字。
4. 在左边加上“下”，则表示“情”的意思，也就是“血液中包含的东西”的含义。

汉语中的“众生”在西夏语中是通过“具有感情的人→有情”的形式表现的，
而“菩萨”的“菩”字也是用这个“情”字来表达的。

汉语中的“菩萨”是梵语中“Bodhisattva”的音译的结果。
而在西夏语中，“菩萨”则是通过“领悟了同情”的形式来表示的。
