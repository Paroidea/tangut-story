# 4. 耳朵

{{< image src="images/chapter-3/ear-bronze.svg" title="汉字金文“耳”" class="char" usefigure="yes" >}}

{{< image src="images/chapter-3/ear-seal.svg" title="汉字小篆“耳”" class="char" usefigure="yes" >}}

汉字的“耳” （{{< image src="images/chapter-3/ear-bronze.svg" title="汉字小篆“耳”" class="char-small" >}}、{{< image src="images/chapter-3/ear-seal.svg" title="汉字小篆“耳”" class="char-small" >}}），原本是模仿耳朵的形状创造的象形字。
“耳”字有很多派生字，比如“圣”（聖）、“声”（聲）等等。

{{< tangut_highlight 
  line1="𗐴"
  title="图9 耳"
>}}

而西夏人为了表示“耳朵”的概念，创造出了一个独特的字形（图9）。
这个字形原本的由来现在已经无从知晓，但西夏字中的“耳”是一个基本字，以它为基础存在多个派生字（图10）。
西夏字中的“圣”或者“声”，虽然与“耳”字完全没有关系，但与耳朵最为紧密关联的“听”却是“耳”的一个派生字。

汉字中的“听”（闻[^闻]）写成“门里有耳”的形式，也就是表现“把耳朵贴在门上”这一状态的会意字。
而在西夏字中，则是在“耳”的左边加上人偏来表示“听”。
这一动词的读音，根据后面接的人称不同，可能会从“你”（niɦ）变化成“貌”（mǐɔɦ）。
变化后的字形写法也要改变，在顶上需要再加一个冠（即“𗔘”）。

{{< tangut_highlight 
  line1="𗐴,𗨉,𗔘,𗑵,𗒨,𘛒" 
  line2="耳,听,（我在）听,铙,䥽,乱"
  title="图10 “耳”的派生字"

非常特别而又有意思的是，两种非常吵闹到刺得耳朵疼的乐器“铙”和“䥽”，
以及表示“平安被打破”的“乱”字，都是从“耳”产生的派生字。
同时，西夏语中“针孔”也是通过“针的耳朵”的形式进行描述的。

[^闻]: 日语中“听”写作“聞く”，对应汉字中的“闻”字。
