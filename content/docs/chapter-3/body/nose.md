# 5. 鼻子

{{< image src="images/chapter-3/nose-seal.svg" title="汉字小篆“鼻”" class="char" usefigure="yes" >}}

汉字的“鼻”，是由意符“自”和音符“畀”（音bì）组合成的形声字。
而在西夏字中，“鼻”则是由肉偏和表示“鼻”的部首“𘦱“组成的（图11）。

{{< tangut_highlight 
  line1="𗮮"
  title="图11 鼻"
>}}

汉字中，由“鼻”产生的派生字有“齅”（音xiù，“嗅”的古字）、“䶊”（音nǜ，鼻出血）、“䶋”（音xī，鼻息、鼾声）等等。
而在西夏字中，也有嗅/闻、鼻涕、芳香、香虫等派生字。

{{< tangut_highlight 
  line1="𘎘,𘎞,𘎡,𘎕" 
  line2="嗅，闻,鼻涕,芳香,（香）虫"
  title="图12 “鼻”的派生字"
>}}

- 鼻 + 施加 = 闻到（香味）
- 鼻 + 浑浊 = 鼻涕
- 鼻 + 喜爱 = 芳香
- 鼻 + 虫 = （有香味的）虫（原书标注了？，可能指含义不明）

另一方面，在西夏文字中，表示身体某个部分，或者身体某部分相关的行为的文字，大多都含有“肉”的部首（图13）。比如：

{{< tangut_highlight 
  line1="𗮯,𗮰,𗮚,𗣪,𗮫,𗮺,𗪦" 
  line2="胁（腋下到肋骨的区域）,肝,尿,排尿,痈/疖,肺,疤"
  title="图13 包含部首“肉”的文字群"
>}}

- 肉 + 手 = 胁
- 肉 + 心 = 肝
- 肉 + 水 = 尿
- 尿 + 人 = 排尿
- 肉 + 恶 = 痈
- 痈 + 人 = 疤
- 肉 + 膨胀 = 肺

{{< tangut_highlight 
  line1="𗮿,𗯋,𗯎,𗯇,←,𗯋,𗯴,𗍫" 
  line2="交合,男生殖器,女生殖器,睾丸,←,男生殖器,下,二"
  title="图13 包含部首“肉”的文字群（续）"
>}}

- 肉 + 二 = 交合
- 肉 + 好 = 男生殖器
- 肉 + 喜爱 = 女生殖器

而最巧妙的则是“睾丸”的字形，对其进行分析就会发现，它是由“男生殖器”、“下”、“二”三个文字取分别取了一部分构成的，也就是说，男生殖器下面的两个物体就是睾丸。

{{< tangut_highlight 
  line1="𗮹,𗮶,𗮨,𗮳" 
  line2="膀胱,脾脏,臀部,“胯骨”的“胯”"
  title="图13 包含部首“肉”的文字群（续）"
>}}

除此之外，“膀胱”、“脾脏”、“臀部”、“胯”等文字中，也包含部首“肉”。

> 《简明夏汉词典》中对“𘎕”的释义为“蛆虫”。
