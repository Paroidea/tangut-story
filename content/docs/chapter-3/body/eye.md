# 2. 眼睛

{{< tangut_highlight
    line1="𗑉" 
    title="图4 眼睛"
>}}

汉字里的“眼睛”，是根据人类的眼睛的形态所创造的象形字，具有非常简单的笔划。
而在西夏文字中（图4），“眼睛”的写法相当复杂。

{{< image src="images/chapter-3/eye-bronze.svg" title="汉字金文“目”" class="char" usefigure="yes" >}}

如图5所示，西夏文字中的“眼睛”是由即作为音符又作为意符的(a)部分加上(b)冠所构成的，
从字形的角度来看，可以认为是形声字，发音为“梅”（mʷε，平声三十三韵[^平声三十三韵]）。

{{< image src="images/chapter-3/section-2_fig-5.svg" title="图5 “目”的派生字" usefigure="yes" >}}

图5(a)的文字表示的是“明显”的含义。
还有一个字(c)跟它类似，只有偏的形态有点细微的差别，表示的是“聪明、有智慧”的意思。
(a)“明”和(c)“聪”两个字是同格的，各自作为基础字都有若干的派生字，而“明”的派生字之一便是“眼睛”。
另一方面，“聪”的派生字中，既有加上水偏形成的(d)和加上石冠形成的(e)。其中的后者(e)表示“玉石”的意思，(d)和(e)两个字组合在一起表示“琉璃”的含义。而加上人偏之后形成的(f)“慧”，也是“聪”的派生字之一。

西夏人把“眼睛”看作是“明显是透明的物体”来理解。

就像汉字中与“目”（眼睛）对应的还有“眼”（眼珠）一样，西夏字中也有“眼”字（图6(g)）。
“眼”字是由部首“目”偏作为旁，然后加上人偏所构成的。
由于这个字形与眼睛的功能有关，因此从它可以派生出(h)“观察”、(i)“审查”（详细调查）等等派生字。

将部首“目”作为偏，将“水”作为旁，意味着把眼睛和水结合在一起，表示(j)“哭泣”的意思。
或者如果以“目”为偏，把旁替换为部首“下”，组成的字意味着“闭上眼睛沉默”的样子，
常被用作汉字中“玄”、“默然”等含义的翻译。
