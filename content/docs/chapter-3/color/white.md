# 2. 白色

{{< image src="images/chapter-3/white-seal.svg" title="汉字小篆“白”" class="char" usefigure="yes" >}}

汉字里“白”的字形，象征着大拇指的指甲的性质，原本是借用了表示大拇指的字“擘”的读音的借用字。

{{< tangut_highlight 
  line1="𗴂"

>}}

西夏字的“白”，则是将“双”（𗴃，读音“庞”，平声五十五韵）作为音符，将旁替换为表示“没有带上任何东西”的“𘥓”得到的形声字（图3）。
这个旁本身可以作为单体字使用，比如像“𗭒𘇥”一样，与“足”一起使用，表示“裸足”的意思。

这里介绍的“白”字，属于西夏语中的“词汇层II”，与缅甸语中的“phyu”（白）为同源词。
同时，在以缅甸语为代表的藏缅语系中，很多语言都有这个词的同源词。

“白”还有一个强调的形式“𗴂”（读音：庞奢奢，phǐon tšhǐě tšhǐě），表示“纯白”。
这个强调形式非常特别，因为“黑”、“红”的强调形式都是像这样加上双音节叠词，而且韵母相同，只是根据色彩的不同，声母有所变化。
与“白色”的“tšh-”相对应，“黑色”是“k-”，“红色”是“tš-”。
这种强调的形式，在彝语、缅甸语中，也都有类似的存在。

隶属于“词汇层I”的“白”，写作“𘙮”（读音：勒，上声三十八韵）。
在名为《月月乐诗》的诗中，与前面的“𗴂”作为对照被使用。

此外，在某个文本中，存在“𗐉𗴂𘅞𘖦𘟨”的一句话，意为“白发而且脸上有皱纹”。
这句话中，“白发”的含义以“发白”的形式进行表现，将状语后置。
西夏语中，涉及到色彩时，正常的顺序都是把色彩的词放在被限定的名词的后面。
