# 6. 特定文字部件表达的含义

西夏文字中，某些特定的文字部件可以用来表现特定的含义，从不同文字间的联系可以非常明晰地看出来。
由此可以得出不同字形在含义上的相互关联，进而能够更加高效地记住这些文字。

比如说，“𘧃”或者“𘠁𘧃”是“装东西”的意思。
带有这个部件的文字，都会在特定的角度上与“装东西”的概念存在关联。

1. 加上“水”偏，表示往里面装水 → “瓶”
2. 加上“袋”偏，表示往袋子里面装 → “袋”
3. 加上“前”偏，表示身体上前面的部分里面的东西 → “胃”
4. 加上“土”偏，表示土里面装的东西 → “坟（土堆）”
5. 加上“死”偏与“土”冠，表示装有死人的地方 → “冢”

这些字形，反映了西夏人有趣的思考方式。
了解了类似这样的字形关联，就能够更加容易地记忆这些字形。
