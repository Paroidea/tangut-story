# 3. 红色

汉字的“赤”（{{< image src="images/chapter-3/red-oracle.svg" title="汉字甲骨文“赤”" class="char-small" >}}），原本是从“大”和“火”字创造出来的。
这是表示“直到火烧得很旺的时候，火焰才会显现出赤色”的含义的会意字。

{{< tangut_highlight 
  line1="𘔚"
  title="图4 红"
>}}

西夏文中的“赤”，则将是将“𘔏”（沙，也就是红土）的旁，替换为“𘦜”（含义不明）形成的派生字（图4）。
其读音为“你”（平声三十六韵），也是属于“词汇层II”的文字。

就像缅甸语中的“ni”（红色）一样，在藏缅语族中有很多词语都是西夏字“红”的同源词。

“红”的强调形“纯红”，则是“𘔚𘔓𘔓”（读音：, neɦ tšıě tšıě）。后面两个文字的旁，与“光”字（𘉍）的旁“𘢒”是一致的。

与“𘔚”对应的，在词汇层I中的“红”字是“𘁻”（读音，適，šıɦ，上声十二韵）。
这个字或许与柿子有某些关联。
在颜色上，会在“𗩨𘁻”（汗色）等词中使用。
将“𘁻”的旁进行替换，得到的派生字“𘁼”与原字读音相同，含义也有所关联，表示“柿子”的意思。
此外，“𘔚”的派生字有“𘔟”，也是“红”的意思。
