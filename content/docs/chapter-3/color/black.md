# 1. 黑色

{{< image src="images/chapter-3/black-bronze.svg" title="汉字金文“黑”" class="char" usefigure="yes" >}}

汉字的“黑”，是由“柬”（表示袋子里有东西）和“火”创造出来的会意字。
表示被火熏黑的含义。

对应的西夏字“黑”（图1），虽然字形的来源还不清楚，但以其为基础诞生了数个派生字（图2）。

{{< image src="images/chapter-3/section-5_fig-2.svg" title="图2 “黑”的派生字" usefigure="yes" >}}

1. 配上“𘡊”冠，就变成了“深”。
表示很深的地方一般看上去都是黑的。
缅甸语中，“nak-”具有“黑”和“深”两种意思，这或许也是从同一种想法而来的。
藏语中，则有从“nag-po~gnag-pa”（黑色）派生出来的“nags”（黑的地方），对应“森林”的含义。
2. 西夏字中，这个“深”字进一步加上“土”偏，就变成了“坑”的含义。
“坑”是很深的，从外面看上去会显示出黑色的样子。
3. 加上“水”偏，就变成了黑水。
这个字或许可以作为一个固有名词来看待，表示水面波澜起伏比较激烈的状态。
根据西夏人的注解，这个字表示“渊水、海水、池、江、沟、河、海里的水，非常深、非常广阔的意思”。
4. 加上“水”冠，似乎表示河流里的沉淀下来的浑浊的物体，也就是“水里的泥”。
5. 加上“人”偏，表示人看上去是黑色的样子，也就是“夜晚”的意思。
6. 最后的“蝌蚪”，也是这个“黑”字加上“虫”偏形成的派生字。
因为“黑色”是“蝌蚪”最大的特征之一。
