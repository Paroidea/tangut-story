# 4. 青色

{{< image src="images/chapter-3/grue-seal.svg" title="汉字小篆“青”" class="char" usefigure="yes" >}}

汉字的“青”，是意符“丹”和音符“生”（变形为“青”）所组成的形声字。

{{< tangut_highlight 
  line1="𗘍"
  title="图5 青"
>}}

西夏字的“青”的造字方式目前尚无定论，但是它的偏与同样表示“青”的文字“𗘚”相同，而它的旁“𘠁𘣮”则不知有何含义（图5）。

“青”的读音是“兀”（ŋʷʉr），也是属于词汇层II的形式，与藏语中“sngon-pa”是同源的。
前面提到的“𗘚”（读音“区”，tshǐuɦ，平声三韵）是它在词汇层I中的形态。

在“青”的基础上加上“水”偏得到的派生字“𗋯”，是以“青”为音符的形声字，表示“潮湿”的意思。

{{< tangut_highlight

“青”也在类似“玄青”（𗘍𘓲）这样的词汇中被使用。
而“𗘓”虽然只是一个单音节词，但从字形上看，它可以被认为是像前面提到的“白”那样的强调形式。
在《掌中珠》中，还有“青龙”、“白虎”这样的涉及色彩的用词，但这些词汇，都只是用了词汇层II里的字形。
