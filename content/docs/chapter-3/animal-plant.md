# 四、表示动物、植物的文字

## 1. 犬

{{< tangut_highlight
    line1="𘋿𘋽,𗘂"

汉字中表示动物的文字大多都是象形字。
与之对应的，在西夏字中使用了多种不同的字形，但不是每个字形的来源都可以被清楚地理解。
首先将“看门狗”的基础上加上“犬”偏的字形，指的是一般的狗（图1）。

就像西夏人所标注的“保护家里的东西”一样，“狗”似乎起到了门卫的作用。
看门狗里的第2个文字与“门”的读音相同，实际上或许就是门的意思。

    title="图2 “猪”（豚、豕）"
>}}

具有“犬”偏的文字。除此之外还有猪（图2）。
除去“犬”偏之后剩下的部分（𘎧）是音符“哇”，原本这个字是部落姓氏专用字，作为表音文字来使用。
从汉语借用的词语“瓦”，就是用这个字形来表示的。

## 4. 羊、骆驼以及其他

{{< tangut_highlight 
  line1="𗂽,𗆎,𘄌𘄃,𘂗𘂞,𗒤𗍻,𗍮,𗒇"
  line2="羊（总称）,羖/雌羊,羊羔,比羊羔大的羊,一岁的羊,两岁的羊,长了六颗牙的羊"
  title="各种“羊”的词语"
>}}

“羊”的总称为“𗂽”。
而羖（雌羊）是“𗆎”，小羊羔是“𘄌𘄃”，稍微大一点的羊是“𘂗𘂞”，再往上的话，“𗒤𗍻”是一岁的羊，“𗍮”是两岁的羊，“𗒇”是长了六颗牙齿的羊等等。
在羊的成长过程中，对应于不同的阶段存在不同的叫法。

而对于骆驼来说，其种类或是别名也并不少。
但它们之间的差别并没有像羊那样被理解的比较透彻。
骆驼的总称是“𘃿𘀤”，“𗶄𗉠”也是骆驼，只用一个字表示的话，“𘀢”也是骆驼。
这些叫法之间究竟有什么差别，现在还没有足够的资料进行判断。

“𘒗𗼺”（齿立）表示长着牙齿的东西，大概指的是成年之后的骆驼。
“𗽒𘋮”表示“峰立”，应该也是骆驼的别名之一。

{{< tangut_highlight 
  line1="𗰻,𗃬,𘃃,𘃁,𗱈"
  line2="恐,怖,豹,熊,虎"
  title="图10 包含“恐怖”作为意符的字"
>}}

而对于豹（𘃃）、熊（𘃁）、虎（𗱈）等动物，西夏人认为它们都非常可怕。
这些动物的名称中都包含有“恐怖”作为意符。

## 5. 禾

{{< image src="images/chapter-3/grain-seal.svg" title="汉字小篆“禾”" class="char" usefigure="yes" >}}

汉字的“禾”（{{< image src="images/chapter-3/grain-seal.svg" title="汉字小篆“禾”" class="char-small" >}}），是象征着水稻的头部下垂的样子的象形字。

{{< tangut_highlight 
  line1="𘌂,𘌐,𘌃"
  line2="图11 禾,图12 米,图13 稻"
  title="图11-13 禾、米、稻"
>}}

“禾”的西夏字也具有与之类似的字形（图11），但并不能说是象形字。
不过，西夏字中的“禾”是单独使用时也具有含义的单体字，在其基础上加上各种不同的文字要素，可以形成很多派生字。

在“禾”的右边加上“人”，就形成了“米”（图12）。
就像前面所说的那样，从“米”可以派生出“粥”、“饭”、“秋”等文字。

另外，在“禾”的右边加上意义不明的“𘠴”，就形成了“稻”字（图12）。
以它为基础，加上不同的偏，可以形成很多不同的文字。

{{< tangut_highlight 
  line1="𗻧,𗋠,𗡄,𘌋"
  line2="大麦,糯,食物,麦"
  title="图14 包含“稻”的派生字"
>}}

这些文字都是比较基本的文字（图14）。

比如“麦”字，是加上了“甜”的旁，把这个意思跟“禾”原本的“倾斜”的含义组合在一起，就成为了“麦”。

在“稻”的偏和旁中间，加上“粉”（𘆩）的旁（𘤕），就成为了“面”（𘌕）。
“五谷”（𘌎）也是从“禾”派生出来的文字，因此在西夏文中经常会出现类似“𗵞𘌎”（谷物）的词语。

{{< tangut_highlight 
  line1="𘌑𘌖,𘌉𘌖"
  line2="粟秫,黍秫"
  title="图15 包含“糯粟”的文字"
>}}

> 译者注：原书中图15里“粟”的西夏字右上角没有一横，但图18里是有的。

与其类似的还有一个字“秫”（𘌖），在图15所示的词语中经常被使用。

{{< tangut_highlight 
  line1="𘎜,𘙔"
  line2="粘,胶"
  line3="加上“鼻”偏,加上“皮”偏"
  title="图16 表示“有粘性物体”的文字"
>}}

在“秫”的字形里把左边的偏去掉，剩下的部分具有“有粘性的东西”的含义。
与“秫”对应的，有“粘”、“胶”等字。

西夏人对“胶”的注解是，“将皮、鹿、角、鱼进行加热，就能得到胶”。
当时的人们似乎会用动物的皮、鹿肉、鱼肉等来制作胶。
其中，“皮”被看作是原材料的代表，以此为基础创作了“胶”字。

## 6. 豆

{{< tangut_highlight 
  line1="𘌙"
  title="图17 豆"
>}}

西夏字中，“豆”字也具有“禾”偏（图17）。

{{< image src="images/chapter-3/bean-bronze.svg" title="汉字金文“豆”" class="char" usefigure="yes" >}}

汉字里的“豆”，原本是将某种容器的形象转用为“豆”的含义创造出来的文字。

{{< tangut_highlight 
  line1="𘌄,𘌇,𘌉,𘌓,𘌑"
  line2="荞麦,穗,黍,糠/穅,粟"
  title="图18 带有“禾”偏的文字"
>}}

将“禾”作为偏，配合其他的意符作为旁，可以创造出多个会意字（图18）。

比如说，“穗”就是“禾”偏加上“尖端、细”组成的。
就像在《说文》中，“穅”指的是“谷物的皮”一样，西夏人也将“禾”偏与“皮”组合在一起来表示“穅”。
