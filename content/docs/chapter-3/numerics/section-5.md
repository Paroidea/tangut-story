# 5. 西夏语中的天干地支

在西夏国，人们也会使用天干地支[^干支]。这些文字页使用复杂且笔划繁多的字形书写的（图4）。

“甲”的西夏字也表示“苗”的含义，而“乙”则表示“芽”的意思。把“已”字加上木冠即可派生得到“甲”字。“丙”和“丁”的西夏字是只有一个含义的特有字，两者都包含“火”的部首，但其字形比较难以区分。
“戊”字似乎本来有“大约”或“左右”的意思，在《同音》中也有“夜”的标注，在“戊夜”一词中表示半夜四点左右的“左右”。此外，在“纯黑”（𗍗𗍇𗍇）一词中也有用到这个字。

“己”的西夏字，是从其“土地”的含义，以及tsị的发音这两个根据判定得到的。“庚”的西夏字有“改变、翻译”的含义，而天干中的“庚”本来就有“变化”的含义，因此把这个字用在天干中，意味着西夏语中是根据其原本的含义选用相应的文字的。“辛”的西夏字意义较难确定。虽然这个字可以指“果物的皮”“选择”等含义，但这与汉语中“辛”字的含义并不一致。

“壬”与“癸”也跟“辛”一样，汉语与西夏语中的含义上并不完全对应。西夏语中的“壬”是“𘋏”（表示“晚”）的音符派生出来的字形，而“癸”字则表示“贮存”的意思。

由于《掌中珠》一书的存在，西夏语中天干的含义很早就已经被判明。但关于字形本来的意义，以及其含义与汉语中是否一致这两点，目前仍然不完全清楚。

西夏语中的十二地支（图5）中，除了不存在的“辰”以外，实际上都对应着十二生肖。寅、巳、戌分别跟虎、蛇、犬使用相同的字形。在西夏语中，不同种类的马会使用不同的文字来表示，类似这样的情况在其他动物中也存在。十二地支的文字，让人感觉很多都从这些含义类似的字中选择一个来用的。

图5中对于表示动物的文字，我选取的都是比较标准而又有统称含义的文字。

而对于虎、蛇和狗，这三种动物都只有一个对应的文字。

[^干支]: 译者注：原文为“十干和十二支”，应该与中国的天干地支是同一个内容。
