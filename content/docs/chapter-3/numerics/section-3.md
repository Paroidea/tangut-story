# 3. 序数词

序数词原本是比较复杂的特别形式的数词，用于表示月份的顺序或家中男子的排行，如长男、次男、三男等。不过，在佛典等文献中，一般使用的序数词都与汉语类似，是在上述基本数词之后加上(p)“第”（本来有“区分、分开”的意思）形成的。
但与汉语不同的是，其顺序截然相反。例如，第一写作(q)“一第”，第二写作(q)“二第”。这因为在西夏语中，限定词位于被限定词之后的规则所致。就像“红花”会写作“花←红”这样的结构一样。这是藏缅语族中的一个古老的形态，在西夏语中得以很好地保存下来。

在譬如佛典卷数的表示中，都是用的是一第、二第、三第这样的形式。

在《孟子》的西夏译本中，有(r)这样的表达，这并不是“十一”，而是“十分之一”的意思，与(s)的用法是不同的。
