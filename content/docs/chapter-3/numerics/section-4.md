# 4. 数词的书写顺序

确实，西夏文字中不管是哪个数词，字形都相当复杂。关于这些字形究竟需要以怎样的顺序书写，目前并没有相关的文献留存。因此，只能用现代的自然书写习惯来推测其笔顺。我个人是按照下面这样的顺序来书写数词的。

上面是从一到十，以及百的书写顺序。其中，“𘢌”（人偏）总是按照“𘠂”，“𘡂”，“𘫼”这样的顺序[^人偏笔顺]书写，因此在图中没有再详细展示（图3）。

[^人偏笔顺]: 译者注：先从上往下写三撇，最后加上一捺。
