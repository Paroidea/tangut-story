# 2. 西夏语的数词使用十进制

西夏语的数词体系，与汉语和日语相同，使用的是十进制。例如，十一是(j)“十”和“一”，二十二是(k)“二”、“十”和“二”，五百是(l)“五”和“百”，像这样直接并列书写就可以。

(m)“百”、“千”、“万”的字形较为复杂。像前面所说的那样，“百”和(n)“殊”的字形非常容易混淆。

在佛典等文献中，还会出现更大的数词，不过(o)“亿”和“兆”这样的数词当然并不常用。
