# 15. 存在否定的文字

将“无”字（𗤋）里右半的旁给取出来，附加到其他文字上，可以表示存在否定。这一点与前面提到的否定偏一样，也是西夏文字的特征之一（图23）。

{{< tangut_highlight 
    line1="𗤋,𘣩,,𘔺,→,𘔲,,𗦫,→,𘊛" 
    line2="无,“无”字的旁,,气,→,尸,,缘,→,譬"
    title="图23 表示存在否定的派生字"
>}}

- 所谓“没有生气的东西”，也就是“尸”。
- 虽然没有直接的关系（缘），但用它来描述事情，就叫做“譬”如。

类似这样的字形构成方法，也很好地反映了西方人的思维方式（图24）。


{{< tangut_highlight 
    line1="𗵒,→,𗒘,,𘘢,→,𗪰,,𘑗,→,𘑘𗢨" 
    line2="金,→,真,,依,→,妻,,山,→,仙人"
    title="图24 字形构成所反映的思维方式"
>}}

- “金子”与“真相”通过顶部的冠相互对应。
- “依靠”加上人偏，就表示依靠的人是“妻子”。
- “山”的一部分换成人，就成为了“仙”。

类似这样的构字方法乍一看与汉字有些相似，但实际上存在根本的区别。 汉字中的“仙”字，是由“亻”（人）作为表意的部分，“山”作为表音的部分构成的形声字。而西夏文字的“仙”当中，是将“山”的字形的一部分替换成“人”所构成的特别的派生字。
