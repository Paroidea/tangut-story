# 2. 《番汉合时掌中珠》

{{< card image="/images/chapter-2/img_37_1.jpg" >}}
《番汉合时掌中珠》
{{< /card >}}

在科兹洛夫探险队第一次的发掘品中，有一本书叫做《番汉合时掌中珠》。这是一本西夏语与汉语对译的词汇集。井上靖的电影《敦煌》中登场的角色赵行德制作的词汇集，就是以这本书为原型的。为了电影《敦煌》，我根据《掌中珠》和被认为是其源头的《番汉语要集》（推测的书名）为原型，制作了一个小道具，作为赵行德创作的词汇集。这个小道具后来在奈良丝绸之路博物馆的丝绸之路邮局里进行了展览，希望后世不要认为这个是真正的文物。

{{< card image="/images/chapter-2/img_37_2.jpg" >}}
铜制印章
{{< /card >}}

真正的《番汉合时掌中珠》是由西夏人骨勒茂才编纂的。因为需求很大，似乎曾经多次再版。原本现存数种，但由俄罗斯的学者伊万诺夫长期持有，至1909年将一部分内容进行了研究并发表论文。
其后，中国方面根据原件的照片，将其内容进行复制，制成了书籍。这样的书籍也有两种以上的版本，其中缺失的部分经由身处爪洼岛的德国研究者在日本公开发表，用了数十年时间，终于整理出大体完整的影印本。
目前一般使用的，都是从各处收集来的复印本。而原本全部的照片，至今尚未完全公开，因此无法完全确认其全貌。

{{< card image="/images/chapter-2/img_38.jpg" >}}
西夏语翻译的《孙子》
{{< /card >}}

在此前访问列宁格勒时，一位名叫克罗科洛夫的学者曾声称将《掌中珠》的资料全部重新整理，并发表自己的研究。但此人已经去世，后来的进展也不为人知。

令人不可思议的是，《掌中珠》明明再版过很多次，之后在中国的发掘中哪怕再发现一页内容也不足为奇。然而，实际情况是到目前为止，连一点相关的残片也没有发现。
