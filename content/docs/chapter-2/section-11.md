# 11. 单体字与合体字

首先，西夏文字中也存在单独使用时也有其特有含义的单体字（图6）。虽然这些基本字的数量较少，但在图7所举的合体字中也可以作为部件存在。

{{< tangut_highlight 
  line1="𘂪,𗸕,𗢨,𘝣,𘕰,𘞗,𗹭,𘘤,𘉋,𘌂" 
  line2="单,半,人,浊,树,种,高,皮,八,稻"
  title="图6 单体字"
>}}

然而，在西夏文字中，除了这些单体字之外，还存在将两个以上的要素组合而成的合体字，合体字在西夏文字中占据了大部分。

合体字有多种类型：

1. 将相同的文字要素组合在一起（图7）。
    {{< tangut_highlight 
    line1="𘂰,𗉋,𘏷" 
    line2="双,集,分"
    line3="两个“单”左右排列,两个“集”左右排列,两个“手”左右排列"
    title="图7 相同文字要素左右排列形成的合体字"
    >}}

2. 以中央的线为中心，左右对称地配置相同要素（图8）。
    {{< tangut_highlight 
    line1="𘆍,𗷻" 
    line2="秤,齐（相等）"
    line3="重量放置在左右，衡量重量的均衡，即为“称量”,相同的东西放置在左右，取得均衡，即为“相等”"
    title="图8 中间有竖线分隔，左右对称的文字"
    >}}

3. 基于偏和旁，冠与底等文字部件，将两者的位置交换形成的一对文字。我将这种类型称为对称文字。
