# 4. 文字组成规则

{{< card image="images/chapter-2/img_41.jpg" >}}
《说文解字》
{{< /card >}}

首先，我想先把西夏文字的组成结构大致说明一下。简而言之，西夏文字是一种表意文字。它的字形中包含了这个文字所表达的含义。
在大多数表意文字中，“山”都会写成类似“{{< image src="images/chapter-2/mountain-bronze.png"  class="char-small" >}}[^mountain-bronze]”的形状，每个字的形状都跟对应的事物的形状有关（象形[^六书]）；或是把“上”写成类似“{{< image src="images/chapter-2/up-seal.png" class="char-small" >}}[^up-seal]”，把“下”写成类似“{{< image src="images/chapter-2/down-seal.png" class="char-small" >}}[^down-seal]”的形状，把事物用抽象的写法进行表示（指事）。
我们所使用的汉字当中最核心的部分，原本都是用这两种方法创造出来的。
其次，为了表示语言中大量的含义（概念），只用那两种手段的话会需要大量的字符，对普通人来说非常难以记忆。
因此，人们转而用已经创造出来的字形（单体字）组合成结合字形（合体字）的方法。比如说，从“山”可以创造出“仙”、“峰”（形声）、“岚”（会意）这些字。其中，“仙”与“峰”当中，文字的字形与其在语言中的发音相关，因此被称为形声字；而对于“岚”来说，是“山”的含义与“风”的含义结合起来，表现“山里面的风”的意思，因此被称为会意字。
事实上，汉字中大部分字都是用这两种方法创造出来的合体字。

比如，从“山”可以造出“仙”或“峰”（形声）这样的字，或造出“嵐”（会意）这样的字。“仙”和“峰”与词语的发音相关而产生，因此称为形声字；而“嵐”是由“山的气”和“风”的含义组合而成的字，因此称为会意字。像这样由两种方法构成的合体字，占了汉字的大部分。

西夏文字的构成原理虽然也是一样的，但首先，西夏文字中表示具体事物的象形字完全不存在。

西夏文字中存在少量的单体字，和大量将各种要素组合在一起得到的合体字，虽然这一点与汉字比较相似，但合体字中各个要素的字形的组合方式却从根本上就不一样。因此从这一点上来说，虽然西夏文字中有一些看上去会让人觉得它是不是在模仿汉字，但纵观全局，基本上都是独立创造出来的字形。

由于这些字形的创造方式反映了西夏人的思维方式，因此我对这一点非常地感兴趣。

在汉字当中，合体字的构成要素有“偏”“旁”和“冠”等等，这些要素并不一定可以作为单体字直接使用。比如说，汉字中的形声字“漢”是由偏旁“氵”（表示水）和音符“𦰩”组成，但作为音符的“𦰩”不会单独使用。因此，在组成文字的要素中，作为部首的不同的字形究竟表达的是什么含义，还有很多没有理清的内容。

[^六书]: 六书，是汉字中造字与用字的原理，具体包括以下六条：

    象形，比如古代文字中的“日”，“月”等等，与物体原本的外形相对应，以此来进行造字的原理。

    指事，比如古代文字中的“上”，“下”等等，把上下这样抽象的概念通过在一条基准线上往上或者往下画线来表达含义的造字原理。

    会意，像“信”这样，把“人”和“言”组合在一起，用他们各自的含义组合成新的含义的造字原理。

    形声，向“江”这样，把表示含义的“水”和表示读音的“工”组合在一起得到派生字的造字原理。

    以上四种皆为造字的原理。

    假借，例如“发出号令”中的“令”字，用在“县令”中一样，由于发音相同，于是借用来使用的原理。

    转注，例如“老”与“考”这样，具有相同的部首，发音也相近，语源存在关联的语言中，将相同的字形转用过来的原理。
    
    以上这两种，是用字的原理。
