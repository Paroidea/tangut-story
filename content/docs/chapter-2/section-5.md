# 5. 汉字与西夏文字

{{< card image="/images/chapter-2/img_43_1.jpg" >}}
西夏文典籍（行书体）
{{< /card >}}

汉字，自甲骨文以来，经过长时期的使用，字体（如篆书、隶书等）不断发生变化，经过长时间的发展与整理，最终形成了楷书体。汉字具有非常久远的生命，因而字数达到了六万数千字之多。

与之对应的，西夏文字在1036年作为西夏国的官方文字而公布了一次，恐怕也像前面说的那样经过了多年的考量，最终在公布一年以前最终完成。西夏文字的字数没有汉字那么多，仅有六千几百字，大约是汉字的十分之一。同时，在随后两百年间，虽然一直在日常生活中使用，字形几乎没有变化。因此，可以说西夏文字整体是以一种有组织的方式创造出来的，不同文字之间也具有非常明显的关联。

{{< card image="/images/chapter-2/img_43_2.jpg" >}}
花纹瓶
{{< /card >}}

虽然说我认为这些错综复杂的字形在被创造出来的时候都有其各自的根据，但现在还没有办法把每个字符背后的原因都理清。

在联想的范畴内，我们可以探索词语背后所蕴含的控制字形的原则，以及创作者所意图表达的关系。

在具体分析各个字形之前，我们首先来阐释一下，从所有文字的整体角度来考虑能够明确发现的原则。

西夏文字的字形之间的相互关联，与汉字一样，不光在意义方面，在发音方面也是极其重要的。但是，如果现在就对西夏文字的发音，即读法，是如何在今天进行复原的过程加以说明的话，会不可避免地使用许多复杂的符号，对于一般读者来说也过于难以理解。因此，不得不将这块内容略去不谈。对于读音，只会在无论如何都需要的地方，用日文假名进行表示[^发音表示]。
