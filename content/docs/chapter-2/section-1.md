# 1. 西夏文字辨识的过程


{{< tangut_highlight line1="𗧐𗇘" line2="bie lhew" line3="西夏文字“解脱”" >}}


西夏王国的历史，如今已经基本上都被理清了。之所以会把这个国家称为幻想中的国家或是谜之国，是因为它在两百多年的时间里占据了河西走廊这一关键地域，却在与成吉思汗率领的蒙古军作战失败后突然灭亡。再加上，它在后世遗留下来的西夏文字，看起来像是汉字，但实际上却是一种未知的死文字。这些原因使得人们在一点时间内对它非常感兴趣。

在这里，有必要简单说明一下这种文字，以及它被世人所认知的过程。

这种文字的存在，本身在很早以前就已经为世人所知，但人们并不知道它就是西夏国的文字。1870年，法国的波拿巴伯爵[^bonaparuto]制作并刊行了居庸关文[^居庸关六体文字]的拓本。所谓居庸关，是从今天北京到八达岭万里长城途中的一个过街塔[^过街式塔]。它原本是作为从北京到西域途中设置的一个关卡。来往的人们会在那里祈求保佑旅途顺利，是一个重要的佛教遗迹。正如过街塔的名字所呈现的那样，居庸关的中间是一个隧道，隧道内左右两侧的墙上刻有六种文字的经文。每一种文字都有较大和较小的尺寸的字形刻写的内容。大字刻写的内容，在东壁是尊胜陀罗尼，在西壁则是如来心陀罗尼的梵文读音转写。而小字刻写的内容，从东往西一次记录了这座塔的由来，以及造塔的过程。根据记录的内容，以前在塔上还有大中小三座佛塔，但现在只有底部的塔台还留存于世。

{{< card image="images/chapter-2/img_31.jpg" >}}
居庸关刻有的六种文字
{{< /card >}}

而居庸关文的六种文字，在上半部分有从左向右书写的有兰扎文字（梵语）和西藏文字（藏语）的内容；而在下面的左半部分，从左往右依次是竖行书写，从左往右换行的巴思巴文（蒙古语）和维吾尔文字（维吾尔语）；下面的右半部分从右往左依次是竖行书写，从右往左换行的汉字（汉语），和另一种夹在维吾尔文字和汉字之间的不明文字。最后这一种文字就是西夏文字，但由于墙壁上刻有的内容本身并没有说明那一种未知的文字是哪种文字，最初大家认为那可能是女真文字。

居庸关建造于1345年，那时西夏国已经灭亡118年了。然而，根据碑文的内容可以判断，在西夏国灭亡之后仍然有西夏的遗民在当地活动。在碑文上刻下的西夏文字是漂亮的楷书，而且字形仍然保留的很好。

直到1882年，法国的东方学家德维利亚[^德维利亚]才将这种文字正确地认定为西夏文字。德维利亚同时也介绍了凉州大云寺里用西夏文撰写的感应塔碑文，他是世界上最早的西夏学学者。而感应塔碑文的实物现在保存在甘肃省武威县的文庙中。

随后到了二十世纪，法国的莫里斯对西夏文的《法华经》进行了研究。他在北京获得了绀纸金泥书写的《妙法莲华经》，并对其开头部分进行了研究，于1904年发表了成果。

莫里斯研究的《法华经》
{{< /card >}}

绀纸金泥书写的西夏经卷的数量非常少，到目前为止只有巴黎的吉美博物馆中收藏的莫里斯的四卷旧藏本、中国宁夏出土的法华经贤首品的残片（在敦煌·西夏王国展中展出）、以及日本私人收藏的一块残片。这些都被认为是相当长的历史的抄写本。

西夏语的文法据说与突厥语类似，存在着一定的难度。关于西夏文的研究，最初阶段几乎未能取得进展。莫里斯去世数年后，西夏研究仍然没有出现重大的突破。

但奇怪的是，西夏语的语法，在不同的文本中的难易度差别很大。《法华经》中的西夏文，在这当中属于比较难的级别。在20世纪初的研究中，基本还没有办法能够理解。

莫里斯之后过了几年，发生了一件对西夏研究产生巨大推动的事件。

{{< card image="images/chapter-2/img_33_1.jpg" >}}
黑水城
{{< /card >}}

1907年至1916年间，俄罗斯的地理学会派遣了科兹洛夫上校[^科兹洛夫]前往蒙古和四川进行探险。科兹洛夫率领一支队伍进入蒙古，偶然间听说在黑水城附近一座大型古城后，对这一座“死城”进行了发掘，结果发现了约300多件用奇特文字书写的文书，以及佛像、佛画、纸币等文物。

在当时，科兹洛夫对西夏国的存在，或者是西夏文字的形态当然是一无所知的。因此他无法对出土文物的价值进行评价，将发掘的文物送到了圣彼得堡（今天的列宁格勒[^列宁格勒]）。他自己继续前往原本的目的地，路过西宁向贵德前进。他在贵德停留期间，收到了来自圣彼得堡的新指令，命令他停止前往四川，掉头北上，重新对黑水城进行调查。俄罗斯科学院对这些出土文物给予了高度评价，认为它们是西夏的重要文物。

在经过了一个多月之后的第二次发掘中，科兹洛夫认真进行了工作，在把佛塔破坏之后，幸运地挖掘到了古代的书库，并从中发现了西夏文的印刷本、抄写本等文物。除了发现2000多本文书以外，还有300多张佛画。这是第一次对西夏的重要文物进行的大量收集。

{{< card image="images/chapter-2/img_34.jpg" >}}
带插图的西夏文佛教经典
{{< /card >}}

黑水城的遗迹在上层是元代修建的，而下层是西夏时代建造的。科兹洛夫应该对这些进行了很好的分辨。

科兹洛夫的发掘对西夏研究起到了极大的推动作用，唤醒了700多年以前灭亡的西夏国，并为解读其文字提供了强有力的契机。

在六年后的1915年，英国人斯坦因[^斯坦因]率领第二次中亚探险队也访问了黑水城。虽然大多都是残片，但斯坦因还是带回了相当多数量的西夏文资料。

{{< card image="images/chapter-2/img_35_1.jpg" >}}
翻译成西夏文的《论语》
{{< /card >}}

斯坦因在之前访问大英博物馆的时候，从当时在那里的西夏研究学者格林斯特德那里看到过很多未整理的资料。很多资料与砂土和毛发的混在一起结成块状，将那些资料仔细地分离出来得到的细小的残片给斯坦因留下了很深的印象。

斯坦因的收藏品，也就是科兹洛夫发掘完之后留下的物品当中，经过调查可以发现，其中很多都与科兹洛夫的藏品存在关联。同一部经典被上下切割的情况也存在，那并非自然破裂，而更像是被人为撕裂的痕迹。

{{< card image="images/chapter-2/img_36.jpg" >}}
翻译成西夏文的《孟子》
{{< /card >}}

我几乎看过所有斯坦因收藏的西夏文献，其中有些内容包含了一些非常珍贵的资料，在此对其内容作一简要介绍。

[^居庸关六体文字]: 公元1345年在从北京通往西域途中的重要地点修建的过街塔（内部为可以通行的隧道）。现在留存下来的主要是，当时建有的三座佛塔下面的底座。在其内部刻有曼荼罗和四天王像等，以及六种不同的文字。文字内容中字体较大的是尊胜陀罗尼和如来心陀罗尼的内容，小字部分记录了造塔的历程。六种文字及其书写顺序可以见上面的图。

[^女真文字]: 金的太祖阿骨打命令完颜希尹模仿契丹文字和汉字创造的女真国字，于1119年发布。二十年后，即1138年，金熙宗完颜亶完成了新字，即女真小字的制定与公布。女真文字中有类似汉字的单体字，还有表音字母存在。

    公元1234年金国灭亡之后，在东北地区仍有少数人继续使用，但到明朝中期（公元15世纪）时已经完全没有人会使用。明代编纂的《华夷译语》中收录了《女真馆来文·杂字》等资料，是女真文研究的重要资料，此外也有一些碑文流传下来。1973年，西安文物管理委员会从孝经碑文的石柱中，发现了金人手写的女真字文书残片。

[^科兹洛夫]: 彼得·库兹米奇·科兹洛夫（1863-1935）

    生于俄罗斯斯摩棱斯克。1884年以军人身份参加普尔热瓦尔斯基的探险队，对西北蒙古、西藏北部等地进行了踏查。其后多次进行探险，在阿尔泰地区、青海省、四川省等地开展调查发掘，尤其是因为他在西夏黑水城遗址，以及乌兰巴托地区的诺彦乌拉匈奴王侯墓进行发掘而为人所知。

[^斯坦因]: 马尔克·奥莱尔·斯坦因（1862-1943）

    出生于布达佩斯，后来取得英国国籍。是一名探险家的同时，也是东方史学者。因其三次进行中亚探险而闻名于世，在和田、尼雅、敦煌、黑水城等地取得丰硕成果。
    
    他对东突厥斯坦古代文明的研究，特别是对敦煌文书的收集，使其为人所知。同时，他还发表了多部学术报告。
