# 14. 带有否定偏的文字

前面介绍到西夏人在造字时，因其独特的想法而创作的造字要素之一，边上否定偏。这一偏旁的使用相当频繁，常用于字形的派生（图21）：

{{< tangut_highlight 
    line1="𗰖,→,𗆁,,𘓔,→,𗆫,,𘙰,→,𗆻,,𘎭,→,𗆒" 
    line2="集,→,散,,忘,→,想起,,同,→,孤,,捷,→,懒"
    title="图21 带有否定偏的派生字"
>}}

- “集”否定之后成了“散”
- “忘”否定之后成了“想起”（忘不了）
- “同”否定之后成了“没有相同”，进而是“孤立的东西”
- “敏捷”否定之后成了“懒惰”

添加在右边的否定偏，则既可以用添加的方法进行派生，也有用替换法进行派生的例子（图22）。

{{< tangut_highlight 
    line1="𗚛,→,𗅆,,𗹑,→,𗄵,,𘙲,→,𗅒" 
    line2="动,→,定,,正,→,不正，邪,,长,→,短"
    title="图22 通过否定偏的置换产生的派生字"
>}}

- “动”的偏旁换成否定之后成了“不动”，也就是“定”
- “正”的偏旁换成否定之后成了“不正”、“邪”
- “长”的偏旁去掉，再在左边加上否定之后成了“短”
