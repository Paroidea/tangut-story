# 3. 党项羌族——西夏文化的根基

西夏王国的统治阶层，是被称为党项羌[^党项羌]的部族。

党项羌族在六世纪末登上历史舞台。当时他们居住在今天青海省东南部、黄河河西一带。东接松州，西接叶护，南邻春桑，北通吐谷浑。他们以游牧为生，饲养牦牛、马、骆驼、羊等牲畜，住在用牦牛尾毛或是羖䍽[^guli]（黑色的羊）毛织成的帐篷中。据史籍记载，党项羌族有细封氏等若干部族。其中拓跋氏最有势力，后来成为西夏王国的支柱。

开元年间（公元713-741年），居住在松州附近的党项羌族，因吐蕃（古代西藏）王朝的压迫，请求迁入唐朝境内。唐玄宗允许他们移居至庆州（今天甘肃省庆阳县附近），并授予他们静边州都督之职。

随后在唐朝中期，西北发生大规模动乱。为了防止党项等部受到吐蕃的煽动，使得动乱进一步扩大，唐代宗命令居住在零州（现在的宁夏灵武县）与盐州（现在的盐地县[^yandi]）的党项人迁移到了银州（现在的陕西省米脂县）以北、夏州（现在的陕西省靖边县）以东的地方。唐朝末年，庆州一带居住的党项人分为了陇山（六盘山）以东活动的东山部，和在银州、夏州之间的平地上居住的平夏部。

唐朝末年，平夏部的大首领拓跋思恭帮助镇压黄巢起义有功，被封为定难军节度使，又受封夏国王，获得银、夏、绥、宥、静五州之地，成为日后西夏建国的重要基础。

虽然党项羌族的演变过程已经比较明晰，但关于党项羌族的起源，也就是西夏国统治阶级的起源，仍然扑朔迷离。据称是鲜卑族系的吐谷浑[^吐谷浑与土族]似乎与羌族保持着长久的友好关系，但吐谷浑也征服并管理了很多的羌族部落，其政策对党项羌族到底有着怎样的影响，党项的拓跋氏、吐谷浑的拓跋氏、以及鲜卑的拓跋氏之间到底有着怎样的关联等等问题，在汉文史料中尽管有着诸多记载，但仍有许多问题悬而未决。

无论如何，党项羌族的势力在十世纪末至十一世纪初逐渐壮大，先由夏州向西推进至灵州，继而进一步西移，抵达了黄河的要地。

[^guli]: 译者注：读音为 gǔ lì (古历)，古汉语词汇，指代山羊或黑色的公羊。
[^党项羌]: 羌族[^羌族]中兴起较晚的一支。其中较大的部族有细封、费听、往利、颇超、米擒、野辞、房当、拓跋等八部。其中拓跋部最为强盛，并建立了西夏国。西夏灭亡后，他们流散各地，与其他民族融合。
[^羌族]: 殷商时期，在现在青海、甘肃、新疆南部、四川西部一带，居住着被称为羌的古代民族。他们在北方的历史舞台上具有重要的地位，其名称自甲骨文时代即见记载。
羌族的部族很多，如先零、烧当等等。从魏晋南北朝时期一直到唐宋朝时期，则有宕昌（陇西一带）、邓至、白兰、党项羌等部族较为活跃。虽然他们大多从事游牧，但也有与汉人一起定居，从事农耕的人。今天在四川西北、阿坝藏族自治州的茂汶羌族自治县[^maowen]等地方仍然居住着十万人左右的羌族人，就是古代羌族的后裔。
[^吐谷浑与土族]: 被称为吐谷浑的部族原本是早年居住在辽东的游牧民，属于鲜卑慕容部的一族。西晋末期，其首领吐谷浑率领部众一千七百余户，沿着蒙古的阴山向西迁徙，到达今甘肃、青海之间地区，并在那里征服羌族诸部，建立政权。早期以游牧为主，但后来占据东西交通要道，重视商业贸易。吐谷浑部族似乎讲的是鲜卑语（属于阿尔泰语系蒙古语族）。吐谷浑人接受了佛教的信仰，也保留了鲜卑族祭祀山川日月的习俗。如今，在以青海省互助土族自治县为中心，以及甘肃省天祝等地居住的土族，被认为是其后裔。
[^maowen]: 译者注：1958年，中华人民共和国将汶川县、理县、茂县三县合并，建立茂汶羌族自治县。1963年恢复原三县建制，茂县仍称茂汶羌族自治县。1987年7月，茂汶羌族自治县撤销，仍置茂县。 
[^yandi]: 译者注：原文为“塩地県”，可能指的是宁夏回族自治区的盐池县。
