# 4. 西夏王国的成立

宋仁宗明道六年（1032年）十月，夏太宗李德明在其五十一岁时去世，其子李元昊继承王国，自称“嵬名吾祖”。“嵬名”是大首领的姓氏，“鬼”与“魏”相通；“吾祖”在西夏语中是“皇帝”的意思。李元昊改革衣冠制度、朝廷礼仪和礼乐制度，并建立官制、兵制[^西夏的监军司]等国家机构，同时整顿首都兴庆府。另一方面，他对宋朝和吐蕃采取对抗政策，1035年，他攻占瓜州、沙州、肃州三州，疆域东至黄河，西到玉门关，南达萧关，北临大沙漠，领土大幅扩张。同时，他掌握了东西交通要道带来的利益。1036年颁布了本国文字，这就是后来所称的西夏文字。

在完成这些准备之后，李元昊[^李元昊]于大庆三年（1038年）十月十一日，在兴庆府南郊设坛举行祭礼，正式举行登基典礼，登上皇帝宝座。第二年（1039年），他派遣使者到宋朝，递交国书，解释了建国称帝的原因，并请求宋朝承认其皇帝称号。

[^李元昊]: 李元昊（1003年-1048年），其祖先曾被唐朝赐姓李。在被宋朝赐姓赵后也用过赵元昊的称谓。宋明道元年（1032年），他被任命为夏州定难军节度使，继承西平王的称号之后，将党项羌族的拓跋氏改名为嵬名氏，把自己的名字改为曩霄[^nangxiao]。1038年建立新兴国家西夏王国，并推行独自的制度。后来被太子宁令哥削去鼻子而死。在位十年。谥号武烈皇帝，庙号景宗。
[^nangxiao]: 译者注：读音 nǎng xiāo，或作曩甯、曩宁。
[^西夏的监军司]: 西夏军的最高统帅称为“监军司”，共设十二个（见第15页附图）:
  左厢神勇，石州祥佑，宥州嘉宁，韦州静塞，西寿保泰，卓啰和南，右厢朝顺，甘州甘肃，瓜州西平，黑水镇燕，白马强镇，黑山威福。
  其中，前面的六军属于“左厢集团军”，剩余的六军属于“右厢集团军”。主要兵力集中于东部地区，各军总人数攻击五十余万人。除这些部队外，还有十万人左右的“擒生军”（俘虏部队）。兴庆灵州有精兵两万五千人负责防卫首都，还有被称为“御园内六班”的七万人，分为三班负责夜间防卫皇宫。
